<Info>
  **Prerequisite:** Your POS device must be [registered in Decoda](/modules/settings/payment-management#register-a-new-device) after hardware setup before it can process transactions.
</Info>

Decoda supports several Ingenico payment devices to help you process in-person payments efficiently and securely. Each device is designed for different use cases and environments.

## Available Devices

### Ingenico Link/2500

<Frame>
  <img src="/images/pos-devices/link_2500.png" alt="Ingenico Link/2500" width="200" />
</Frame>

**Best for:** Mobile and portable payment processing

- Wireless connectivity (WiFi)
- Battery-powered for mobility
- Compact and lightweight
- Perfect for providers who move between rooms or locations

<Card title="Link/2500 Setup Guide" icon="arrow-right" href="/modules/settings/pos-devices/link-2500">
  Learn how to set up and use the Link/2500
</Card>

### Ingenico Lane/7000

<Frame>
  <img src="/images/pos-devices/lane_7000.png" alt="Ingenico Lane/7000" width="200" />
</Frame>

**Best for:** Front desk and fixed-location payments

- Large color touchscreen
- Both WiFi and Ethernet connectivity
- Advanced signature capture
- Ideal for high-volume front desk operations

<Card title="Lane/7000 Setup Guide" icon="arrow-right" href="/modules/settings/pos-devices/lane-7000">
  Learn how to set up and use the Lane/7000
</Card>

### Ingenico Lane/3000

<Frame>
  <img src="/images/pos-devices/lane_3000.png" alt="Ingenico Lane/3000" width="200" />
</Frame>

**Best for:** Basic fixed-location payments

- Ethernet connectivity
- Compact design
- Cost-effective solution
- Perfect for smaller practices or secondary payment locations
