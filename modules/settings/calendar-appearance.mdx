<Info>
  **Navigation:** Go to **Settings > Calendar Appearance** to access these settings.
</Info>

Control the visual style of appointments on your calendar. You can change how each appointment status is displayed and add icons that highlight important details at a glance -- like whether a patient paid a booking fee, completed their forms, or was scheduled by AI.

## Appointment Status Styles

Each appointment status can have its own visual style on the calendar. This helps your team quickly tell the difference between confirmed, pending, cancelled, and other appointment states just by looking at the calendar.

The available statuses are:

| Status | Description |
|--------|-------------|
| **Reserved** | A time slot has been held for the patient, but the booking fee has not been paid. |
| **Pending** | The appointment is scheduled but has not been confirmed by the patient. |
| **Confirmed** | The patient has confirmed the appointment. |
| **Completed** | The appointment is finished. |
| **No-show** | The patient did not show up for their appointment. |
| **Cancelled** | The appointment has been cancelled. |

### Style Variants

For each status, choose one of three visual styles:

- **White** -- A white bar along the left edge of the appointment block.
- **Translucent** -- A semi-transparent background using the service color.
- **Opaque** -- A solid, fully colored background using the service color.

A live preview appears next to each status so you can see how it will look on the calendar before saving.

### Change Status Styles

<Steps>
  <Step title="Open Calendar Appearance">
    Go to **Settings > Calendar Appearance**.
  </Step>
  <Step title="Choose a Style for Each Status">
    Under **Appointment Status Styles**, click **White**, **Translucent**, or **Opaque** for each status. The preview on the right updates immediately.
  </Step>
  <Step title="Save Your Changes">
    Click **Save changes** at the top of the section. The button is disabled when there are no unsaved changes.
  </Step>
</Steps>

## Appointment Icons

Appointment icons are small visual markers that appear on calendar events. They give your team a quick way to see important information without opening the appointment -- such as whether the patient is new, checked in, or booked online.

There are seven icon types available:

| Icon Type | What It Means |
|-----------|---------------|
| **Booking Fee Paid** | The patient has paid the booking fee. |
| **New Patient** | This is the patient's first appointment. |
| **Scheduled by AI** | The appointment was booked by an AI assistant. |
| **Form Completed** | The patient has completed their required forms. |
| **Form Not Completed** | The patient has not completed their required forms. |
| **Checked In** | The patient has checked in for the appointment. |
| **Online Booking** | The patient scheduled the appointment themselves online. |

Each icon type can only be used once. Once all seven types are configured, the **Add Icon** button is disabled.

### Add an Icon

<Steps>
  <Step title="Open Calendar Appearance">
    Go to **Settings > Calendar Appearance** and scroll down to **Appointment Icons**.
  </Step>
  <Step title="Click Add Icon">
    Click the **Add Icon** button in the top right of the icons section.
  </Step>
  <Step title="Select an Event Type">
    Choose the event type from the list. Types that already have an icon are grayed out.
  </Step>
  <Step title="Choose an Icon Display">
    Click the icon display button to pick how the icon looks. You can choose either:
    - **Color** -- A colored circle. Use the color picker to select any color.
    - **Emoji** -- An emoji from the built-in categories (Status, Documents, People, Actions, Medical).

    Picking a color clears any selected emoji, and picking an emoji clears the color.
  </Step>
  <Step title="Save">
    Click **Create Icon**. The new icon immediately appears in the list and on matching calendar events.
  </Step>
</Steps>

### Edit an Icon

1. In the **Appointment Icons** section, click the edit button (pencil icon) next to the icon you want to change.
2. Update the color or emoji. The event type cannot be changed after creation.
3. Click **Save Changes**.

### Delete an Icon

1. Click the delete button (trash icon) next to the icon you want to remove.
2. Confirm the deletion when prompted. This removes the icon from all calendar events of that type.
