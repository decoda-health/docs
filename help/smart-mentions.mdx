## Overview

Smart Mentions allows you to quickly reference information throughout Decoda by typing @ in any text field. This includes patient information, practice details, saved snippets, and more.

## Types of Smart Mentions

### Practice Information

- **Practice Name**: Type `@tenant_name` to insert your practice name
- **Custom Saved Content**: Access any custom saved snippets your practice has created

### Patient Information

- **Names**: Access first name, last name, or full name
- **Payment Details**: Insert payment links or outstanding balances
- **Dates**: Reference patient creation dates
- **ID**: Include patient identification numbers

### Assistant Information

Assistant-specific information can be referenced when configuring AI responses and templates.

## How to Create Custom Snippets

You can create your own reusable text snippets that can be accessed via Smart Mentions:

1. Navigate to Settings > Suggestions (or press `âŒ˜K` (Mac) / `Ctrl+K` (Windows) and search for "Suggestions")
2. Click the + button to create a new suggestion
3. Give your snippet a label (this is what you'll type after @)
4. Enter the content you want to insert when this snippet is used
5. Save your snippet

Your custom snippets will now appear alongside other suggestions when you type @ in any text field.

## How to Use Smart Mentions

1. In any text field, type the `@` symbol
2. Begin typing to see matching suggestions
3. A preview card will appear showing the actual content
4. Select the desired information to insert

## Best Practices

- Use Smart Mentions to ensure accuracy in your communications
- Save time by quickly inserting common information
- Reduce errors by using auto-complete instead of manual typing
- Preview cards help verify you're inserting the correct information
- Create custom saved snippets for frequently used text
