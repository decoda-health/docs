# Call Management

The Call page manages phone call history and recordings. View call logs, access voicemails, listen to recordings, and review AI-generated call summaries.

## View Call History

<Steps>
  <Step title="Navigate to Calls">
    Go to **Communications** â†’ **Calls**
  </Step>
  <Step title="Browse call list">
    See all inbound and outbound calls
  </Step>
  <Step title="Filter calls">
    By date, direction, or status
  </Step>
</Steps>

## Call Details

Each call entry shows:
- Date and time
- Patient name (if identified)
- Phone number
- Duration
- Direction (inbound/outbound)
- Recording status

## Listen to Recording

<Steps>
  <Step title="Click call entry">
    Opens call details
  </Step>
  <Step title="Click play">
    Audio player appears
  </Step>
  <Step title="Listen">
    Play, pause, seek through recording
  </Step>
</Steps>

<Info>
Call recordings may have retention limits based on tenant settings.
</Info>

## AI Call Summary

For calls handled by the AI assistant:

<Steps>
  <Step title="Open call details">

  </Step>

  </Step>
  <Step title="View transcript">

## Call Actions

| Action | Description |
|--------|-------------|
| **Call Back** | Initiate return call |
| **Link to Patient** | Associate with patient record |
| **Add Note** | Add internal notes about call |
| **Download** | Download recording file |

## Troubleshooting

| Issue | Solution |
|-------|----------|
| No recordings | Check recording is enabled |
| Can't play audio | Check browser audio permissions |
| Missing calls | Check date filter range |
