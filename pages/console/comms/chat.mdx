# Chat & Messaging

The Chat page provides a comprehensive patient messaging interface. Send and receive SMS messages, use templates, and manage conversations.

## View Conversations

<Steps>
  <Step title="Navigate to Chat">
    Go to **Communications** â†’ **Chat**
  </Step>
  <Step title="Browse conversations">
    Left panel shows all patient conversations
  </Step>
  <Step title="Select conversation">
    Click to view message history
  </Step>
</Steps>

## Send Message

<Steps>
  <Step title="Select conversation">
    Or start new by searching patient
  </Step>
  <Step title="Type message">
    In the message input at bottom
  </Step>
  <Step title="Send">
    Click send or press Enter
  </Step>
</Steps>

## Use Message Templates

<Steps>
  <Step title="Click template icon">
    In the message input area
  </Step>
  <Step title="Select template">
    Choose from available templates
  </Step>
  <Step title="Customize">
    Edit the message if needed
  </Step>
  <Step title="Send">
    Template is sent to patient
  </Step>
</Steps>

## Bulk Messaging

<Steps>
  <Step title="Click Bulk Message">
    In the header area
  </Step>
  <Step title="Select recipients">
    Filter and select patients
  </Step>
  <Step title="Compose message">
    Write or select template
  </Step>
  <Step title="Send">
    Messages queued for delivery
  </Step>
</Steps>

<Warning>
Bulk messages may be subject to opt-out preferences. Patients who opted out won't receive messages.
</Warning>

## Conversation Status

| Status | Description |
|--------|-------------|
| **Unread** | New messages from patient |
| **Read** | Messages have been viewed |
| **Archived** | Conversation is archived |

## Attach Media

<Steps>
  <Step title="Click attachment icon">
    Opens file picker
  </Step>
  <Step title="Select file">
    Images or documents
  </Step>
  <Step title="Send">
    Media sent to patient
  </Step>
</Steps>

## Search Messages

Use the search bar to find:
- Patient name
- Phone number
- Message content

## Troubleshooting

| Issue | Solution |
|-------|----------|
| Message not sending | Check patient phone is valid |
| No conversations | Check location filter |
| Opt-out patient | Patient has unsubscribed from SMS |
