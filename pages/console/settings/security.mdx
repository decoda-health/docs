# Security Settings

Configure security policies and access controls for your practice.

## Password Policy

- Minimum length
- Required complexity
- Expiration period
- History (prevent reuse)

## Session Settings

- Session timeout
- Maximum concurrent sessions
- Remember me duration

## Two-Factor Authentication

- Require for all users
- Require for admins only
- Optional for users

## IP Restrictions

Limit access to specific IP addresses or ranges.

## Audit Logging

View security-related events:
- Login attempts
- Password changes
- Permission changes
- Data exports
