# Suggestions Page - User Actions Guide

## Overview

The Suggestions page manages text shortcuts and saved responses. Create quick text snippets that can be inserted anywhere by typing "@" followed by the suggestion label.

---

## Available Actions

### 1. View All Suggestions

**Purpose:** Browse saved text shortcuts.

**Step-by-Step Instructions:**

1. **Navigate to Suggestions**
   - Settings → Suggestions

2. **View suggestions list**
   - Left sidebar

3. **Click to select**
   - Details in center panel

---

### 2. Create New Suggestion

**Purpose:** Add new text shortcut.

**Step-by-Step Instructions:**

1. **Click "+" button**
   - Top right of sidebar

2. **Center panel shows form**

3. **Enter label**
   - The trigger text

4. **Enter content**
   - The text to insert

5. **Save suggestion**

---

### 3. Edit Suggestion

**Purpose:** Modify existing suggestion.

**Step-by-Step Instructions:**

1. **Select suggestion from list**

2. **View/edit in center panel**

3. **Modify label or content**

4. **Save changes**

---

### 4. Delete Suggestion

**Purpose:** Remove a suggestion.

**Step-by-Step Instructions:**

1. **Select suggestion**

2. **Click delete button**

3. **Confirm deletion**

---

### 5. Search Suggestions

**Purpose:** Find specific suggestion.

**Step-by-Step Instructions:**

1. **Type in search box**
   - Below sidebar header

2. **Results filter**
   - By label match

---

## Using Suggestions

To use a saved suggestion:

1. In any text input
2. Type "@"
3. Select from dropdown
4. Or type label to filter
5. Press Enter to insert

---

## Suggestion Structure

| Field | Purpose |
|-------|---------|
| Label | The shortcut trigger (e.g., "greeting") |
| Content | The full text to insert |

---
