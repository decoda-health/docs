# Locations Page - User Actions Guide

## Overview

The Locations page manages practice locations. Staff can add, edit, and configure multiple physical or virtual locations with their own hours, staff, and settings.

---

## Available Actions

### 1. View All Locations

**Purpose:** See all practice locations.

**Step-by-Step Instructions:**

1. **Navigate to Locations**
   - Settings → Locations

2. **View location list**
   - Location name
   - Address
   - Phone
   - Status (active/inactive)

---

### 2. Add New Location

**Purpose:** Create a new practice location.

**Step-by-Step Instructions:**

1. **Click "Add Location"**

2. **Enter location details:**
   - **Name:** Display name
   - **Phone:** Location phone number
   - **Address:** Full street address
   - **City, State, ZIP**
   - **Time Zone:** Location's time zone

3. **Configure hours:**
   - Set hours for each day
   - Mark closed days

4. **Save location**

---

### 3. Edit Location

**Purpose:** Modify location information.

**Step-by-Step Instructions:**

1. **Find location in list**

2. **Click location or edit icon**

3. **Modify details**
   - Update address, phone
   - Change hours
   - Adjust settings

4. **Save changes**

---

### 4. Set Operating Hours

**Purpose:** Configure when location is open.

**Step-by-Step Instructions:**

1. **Edit location**

2. **Find hours section**

3. **For each day:**
   - Set open time
   - Set close time
   - Or mark as closed

4. **Configure breaks (optional)**
   - Lunch breaks
   - Other closed periods

5. **Save changes**

---

### 5. Set Time Zone

**Purpose:** Configure location's time zone.

**Step-by-Step Instructions:**

1. **Edit location**

2. **Find time zone setting**

3. **Select time zone**
   - E.g., "America/New_York"
   - E.g., "America/Los_Angeles"

4. **Save changes**
   - All times adjusted

---

### 6. Configure Address

**Purpose:** Set location's physical address.

**Step-by-Step Instructions:**

1. **Edit location**

2. **Enter address fields:**
   - Street address line 1
   - Street address line 2 (optional)
   - City
   - State/Province
   - ZIP/Postal code
   - Country

3. **Verify on map (if shown)**

4. **Save changes**

---

### 7. Set Location Phone

**Purpose:** Configure contact number.

**Step-by-Step Instructions:**

1. **Edit location**

2. **Enter phone number**
   - Primary contact number
   - Formatted automatically

3. **Save changes**

---

### 8. Configure Review Link

**Purpose:** Set external review platform URL.

**Step-by-Step Instructions:**

1. **Edit location**

2. **Find review link field**

3. **Enter URL**
   - Google reviews link
   - Yelp link
   - Other platform

4. **Save changes**
   - Used in review requests

---

### 9. Deactivate Location

**Purpose:** Disable without deleting.

**Step-by-Step Instructions:**

1. **Find location**

2. **Toggle active status**
   - Or click "Deactivate"

3. **Confirm**
   - Location hidden
   - Data preserved

---

### 10. Delete Location

**Purpose:** Permanently remove location.

**Step-by-Step Instructions:**

1. **Find location**

2. **Click "Delete"**

3. **Review warnings**
   - Check for dependent data

4. **Confirm deletion**

---

### 11. Set Default Location

**Purpose:** Make a location the default.

**Step-by-Step Instructions:**

1. **Find location**

2. **Click "Set as Default"**
   - Or toggle in edit

3. **Confirm**
   - New default set

---

### 12. Assign Staff to Location

**Purpose:** Control who works at location.

**Step-by-Step Instructions:**

1. **Edit location**

2. **Find staff assignment**

3. **Select staff members**
   - Check boxes
   - Or "Assign All"

4. **Save changes**

---

## Location Fields

| Field | Description |
|-------|-------------|
| Name | Display name |
| Address | Physical address |
| Phone | Contact number |
| Time Zone | Local time zone |
| Hours | Operating schedule |
| Review Link | External review URL |

---

## Multiple Locations

When practice has multiple locations:
- Users can be assigned to specific locations
- Appointments are location-specific
- Reports can filter by location
- Separate operating hours per location

| Issue | Solution |
