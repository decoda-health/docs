# Notifications Page - User Actions Guide

## Overview

The Notifications page allows users to configure their personal notification preferences. Control which alerts you receive via SMS and Email, enable location filtering, and manage bulk notification settings.

---

## Available Actions

### 1. View Current Preferences

**Purpose:** See enabled notifications.

**Step-by-Step Instructions:**

1. **Navigate to Notifications**
   - Settings → Notifications

2. **View preference table**
   - Rows: Alert types
   - Columns: Communication methods

3. **Blue toggle = enabled**

---

### 2. Toggle Individual Notification

**Purpose:** Enable/disable specific alert+method.

**Step-by-Step Instructions:**

1. **Find alert type row**

2. **Find method column (SMS/Email)**

3. **Click toggle switch**
   - Saves automatically

---

### 3. Enable All Notifications

**Purpose:** Turn on all alerts.

**Step-by-Step Instructions:**

1. **Locate Bulk Management section**

2. **Click "Enable All" button**
   - All toggles turn on

3. **Button changes to "Disable All"**

---

### 4. Disable All Notifications

**Purpose:** Turn off all alerts.

**Step-by-Step Instructions:**

1. **Click "Disable All" button**

2. **All toggles turn off**

3. **Button changes to "Enable All"**

---

### 5. Toggle All for Method

**Purpose:** Enable/disable all alerts for SMS or Email.

**Step-by-Step Instructions:**

1. **In "By Communication Method" section**

2. **Find method card (SMS/Email)**

3. **Click toggle**

4. **All alerts for that method toggle**

---

### 6. Toggle Section

**Purpose:** Enable/disable all alerts in category.

**Step-by-Step Instructions:**

1. **Find section header**
   - e.g., "Patient Alerts"

2. **Click header toggle**
   - Per-method toggles

3. **All section alerts toggle**

---

### 7. Enable Location Filter

**Purpose:** Only receive location-specific alerts.

**Step-by-Step Instructions:**

1. **In Bulk Management section**

2. **Find "Location Filter" toggle**

3. **Enable**
   - Only alerts from your location

4. **Disable for all locations**

---

## Alert Sections

| Section | Alert Types |
|---------|-------------|
| Patient | New patient, check-in, forms |
| Appointment | Booking, cancellation, reminders |
| Billing | Payments, invoices, failed charges |
| Communication | Missed calls, new messages |
| System | Updates, maintenance |

---

## Communication Methods

| Method | Delivery |
|--------|----------|
| SMS | Text to your phone number |
| Email | Email to your address |

---

## Location Filter

When enabled:
- Only receive alerts from your current location
- Applies to all enabled notifications
- Works for location-specific alert types

---

## Contact Information

Shows your:
- Phone number (for SMS)
- Email address (for Email)

Update in Profile settings if incorrect.

## Troubleshooting

| Issue | Solution |
|-------|----------|
| Not receiving SMS | Check phone number in profile |
| Not receiving email | Check email in profile |
| Toggle not saving | Refresh page, try again |
| No notifications enabled first | Enable before setting location filter |
