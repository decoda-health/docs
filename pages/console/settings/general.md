# General Settings Page - User Actions Guide

## Overview

The General Settings page contains core practice configuration options including business information, branding, contact details, and default settings that affect the entire practice.

---

## Available Actions

### 1. Update Practice Name

**Purpose:** Change the display name of your practice.

**Step-by-Step Instructions:**

1. **Locate the Practice Name field**

   - In the "Business Information" section
   - Shows current name

2. **Click to edit**

   - Click in the text field
   - Current name becomes editable

3. **Enter new name**

   - Delete existing text
   - Type new practice name
   - E.g., "Smith Family Dental"

4. **Save changes**
   - Click "Save" button
   - Changes apply immediately

---

### 2. Update Practice Logo

**Purpose:** Upload or change your practice logo.

**Step-by-Step Instructions:**

1. **Locate the Logo section**

   - Shows current logo or placeholder
   - "Upload" or "Change" button nearby

2. **Click to upload**

   - Click "Upload Logo" or current logo
   - File picker opens

3. **Select image file:**

   - Choose PNG, JPG, or SVG
   - Recommended: 200x200px minimum
   - Transparent background preferred

4. **Upload and preview**

   - Image uploads
   - Preview shows how it will appear

5. **Save changes**
   - Click "Save"
   - Logo updates across platform

---

### 3. Update Contact Information

**Purpose:** Set practice phone, email, and address.

**Step-by-Step Instructions:**

1. **Locate Contact section**

   - Shows phone, email, address fields

2. **Update phone number:**

   - Click phone field
   - Enter primary practice number
   - Format: (555) 123-4567

3. **Update email:**

   - Click email field
   - Enter main practice email
   - Used for patient communications

4. **Update address:**

   - Street address
   - City, State, ZIP
   - Country

5. **Save all changes**
   - Click "Save" button

---

### 4. Set Default Time Zone

**Purpose:** Configure the practice's primary time zone.

**Step-by-Step Instructions:**

1. **Locate Time Zone setting**

   - Dropdown selector
   - Shows current time zone

2. **Click to open dropdown**

   - List of time zones appears
   - Grouped by region

3. **Select your time zone**

   - E.g., "America/New_York"
   - E.g., "America/Los_Angeles"

4. **Save changes**
   - Time zone affects:
     - Appointment scheduling
     - Reminders
     - Reports

---

### 5. Configure Operating Hours

**Purpose:** Set when the practice is open.

**Step-by-Step Instructions:**

1. **Locate Hours section**

   - Shows days of the week
   - Start/end times for each day

2. **Set hours for each day:**

   - Select start time (e.g., 8:00 AM)
   - Select end time (e.g., 5:00 PM)
   - Or mark as "Closed"

3. **Mark closed days:**

   - Toggle day off
   - Or select "Closed" option
   - Typically weekends

4. **Save hours**
   - Click "Save"
   - Hours display to patients

---

### 6. Configure Holiday Closures

**Purpose:** Mark days the practice is closed.

**Step-by-Step Instructions:**

1. **Locate Holidays section**

   - May be separate tab or section
   - Shows configured holidays

2. **Add holiday:**

   - Click "Add Holiday"
   - Enter holiday name
   - Select date(s)

3. **Configure holiday:**

   - Single day or date range
   - All locations or specific
   - Message to patients

4. **Remove holiday:**
   - Click delete on holiday
   - Confirm removal

---

### 7. Set Default Appointment Duration

**Purpose:** Configure standard appointment length.

**Step-by-Step Instructions:**

1. **Locate duration setting**

   - Dropdown or number input
   - Shows current default (e.g., 30 min)

2. **Select new duration:**

   - 15 minutes
   - 30 minutes
   - 45 minutes
   - 60 minutes
   - Custom

3. **Save setting**
   - Used for new services
   - Existing services unchanged

---

### 8. Configure Required Patient Fields

**Purpose:** Set which fields are required when creating new patients.

**Step-by-Step Instructions:**

1. **Locate Patient Settings section**

   - Scroll to the "Patient Settings" section
   - Located after Navigation section

2. **Configure Required Patient Fields:**

   - Click the "Select required fields" dropdown
   - Search and select fields to require:
     - **Email Address** - Patient's email address
     - **Phone Number** - Primary contact number
     - **Date of Birth** - Patient's birth date
     - **External ID** - Your system's patient ID
     - **Address** - Complete address (street, city, state, zip)
     - **Gender** - Patient's gender
     - **First Name** - Patient's first name
     - **Last Name** - Patient's last name
     - **Lead Source** - How the patient was acquired (e.g., Instagram, Facebook, TikTok, Google, Referral)

3. **Remove required fields:**

   - Click the × button on any selected field badge
   - Field is removed from required list

4. **Changes save automatically**
   - Settings auto-save after 1 second
   - Applied to new patient creation forms
   - Existing patients unchanged

---

### 9. Configure Booking Settings

**Purpose:** Set rules for appointment booking.

**Step-by-Step Instructions:**

1. **Locate Booking Settings**

2. **Configure options:**

   - **Minimum notice:** Hours before appointment
   - **Maximum advance:** How far ahead to book
   - **Buffer time:** Time between appointments
   - **Overbooking:** Allow or prevent

3. **Save settings**
   - Affects online and staff booking

---

### 10. Set Practice Description

**Purpose:** Add text describing your practice.

**Step-by-Step Instructions:**

1. **Locate Description field**

   - Large text area
   - May support rich text

2. **Enter description:**

   - Practice specialties
   - Services offered
   - Philosophy or approach

3. **Save description**
   - May appear on patient portal
   - Used in communications

## Best Practices

1. **Keep contact info current** - Patients rely on this
2. **Use high-quality logo** - Appears everywhere
3. **Verify time zone** - Wrong TZ causes scheduling issues
4. **Review hours regularly** - Update for holidays
5. **Test changes** - View patient-facing areas after changes

## Troubleshooting

| Issue                 | Solution                   |
| --------------------- | -------------------------- |
| Logo not uploading    | Check file size/format     |
| Hours not saving      | Verify times don't overlap |
| Time zone wrong       | Clear cache after changing |
| Changes not appearing | Refresh page               |
