# Categories Page - User Actions Guide

## Overview

The Categories page manages categories for organizing services, items, and stock. Categories help group related items for easier navigation and reporting.

---

## Available Actions

### 1. View All Categories

**Purpose:** Browse existing categories.

**Step-by-Step Instructions:**

1. **Navigate to Categories**
   - Settings → Categories

2. **View category table**
   - Name
   - Items count
   - Actions

---

### 2. Create Category

**Purpose:** Add new category.

**Step-by-Step Instructions:**

1. **Click "Create Category"**
   - Top right

2. **Enter category name**

3. **Configure options:**
   - Color
   - Description

4. **Click "Create"**

5. **Category appears in table**

---

### 3. Edit Category

**Purpose:** Modify existing category.

**Step-by-Step Instructions:**

1. **Find category in table**

2. **Click edit action**

3. **Modify details**

4. **Save changes**

---

### 4. Delete Category

**Purpose:** Remove a category.

**Step-by-Step Instructions:**

1. **Find category in table**

2. **Click delete action**

3. **Confirm deletion**

4. **Items become uncategorized**

---

## Category Usage

Categories organize:
- Services
- Products/Items
- Stock

---

## Empty State

If no categories:
- Shows empty message
- "Add Your First Category" button
- Dashed border card
