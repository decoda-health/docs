## User Roles

| Role | Access Level |
|------|-------------|
| **Admin** | Full access |
| **Manager** | Most features, limited settings |
| **Provider** | Clinical features |
| **Staff** | Basic operations |

## Deactivate User

<Steps>
  <Step title="Select user">
    Click on user row
  </Step>
  <Step title="Click Deactivate">
    Confirm action
  </Step>
</Steps>

<Warning>
Deactivated users cannot log in but their history is preserved.
</Warning>
