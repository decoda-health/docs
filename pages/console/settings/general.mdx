# General Settings

Configure basic practice information, branding, and general preferences.

## Practice Information

- **Practice Name**: Display name across the platform
- **Legal Name**: For invoices and legal documents
- **Phone Number**: Main contact number
- **Email**: Primary contact email
- **Website**: Practice website URL

## Address

- Street address
- City, State, ZIP
- Country

## Timezone

Set the default timezone for the practice. Affects:
- Appointment display times
- Report date ranges
- Automated message timing
