# AI Scribe Settings

Configure how the AI Scribe generates clinical notes from recordings.

## Note Templates

<Steps>
  <Step title="Select template">
    SOAP, H&P, or custom
  </Step>
  <Step title="Configure sections">
    Which sections to include
  </Step>
  <Step title="Set defaults">
    Default template per service
  </Step>
</Steps>

## Transcription Settings

- **Language**: Primary language
- **Medical terminology**: Specialty-specific terms
- **Speaker identification**: Multiple speakers
- **Timestamps**: Include in transcript

## Note Generation

- **Auto-generate**: Create note automatically
- **Require review**: Mandate provider review
- **Signature workflow**: Electronic sign-off
