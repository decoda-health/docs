# Inbound AI Assistant

Configure the AI assistant that handles incoming phone calls.

## View Assistants

<Steps>
  <Step title="Navigate to Inbound Assistants">
    Settings → Assistants → Inbound
  </Step>
  <Step title="View assistant cards">
    Grouped by location/type
  </Step>
  <Step title="Search assistants">
    Find specific assistant
  </Step>
</Steps>

## Configure Assistant

<Steps>
  <Step title="Select assistant">
    Click to expand and edit
  </Step>
  <Step title="Set voice">
    Choose voice style
  </Step>
  <Step title="Configure scripts">
    Customize responses
  </Step>
  <Step title="Set behavior">
    Call handling rules
  </Step>
  <Step title="Save">
    Changes apply immediately
  </Step>
</Steps>

## Settings

| Setting | Description |
|---------|-------------|
| **Name** | Assistant identifier |
| **Voice** | Voice selection |
| **Scripts** | Custom call scripts |
| **Behavior** | Response configuration |
