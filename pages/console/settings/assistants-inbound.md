# Inbound Assistant Page - User Actions Guide

## Overview

The Inbound Assistant page manages AI phone assistants that handle incoming calls. Configure assistant behavior, scripts, and settings for automated call handling.

---

## Available Actions

### 1. View All Assistants

**Purpose:** Browse inbound call assistants.

**Step-by-Step Instructions:**

1. **Navigate to Inbound Assistants**
   - Settings → Assistants → Inbound

2. **View assistant cards**
   - Grouped by location/type

3. **Search to filter**

---

### 2. Select Assistant Group

**Purpose:** Expand group to see details.

**Step-by-Step Instructions:**

1. **Click on assistant card**

2. **Group expands**
   - Shows all assistants in group

3. **Click again to collapse**

---

### 3. Edit Assistant

**Purpose:** Configure assistant settings.

**Step-by-Step Instructions:**

1. **Select assistant from group**

2. **Settings panel opens**

3. **Configure:**
   - Name
   - Voice
   - Scripts
   - Behavior

4. **Save changes**

---

### 4. Search Assistants

**Purpose:** Find specific assistant.

**Step-by-Step Instructions:**

1. **Type in search box**

2. **Results filter**
   - By name or group

---

## Assistant Groups

Assistants grouped by:
- Location
- Phone number
- Purpose

---

## Default Assistant

One assistant can be marked as default:
- Handles calls without specific routing
- Shown with special indicator

---

## Configuration Options

| Setting | Purpose |
|---------|---------|
| Name | Assistant identifier |
| Voice | Voice selection |
| Scripts | Custom call scripts |
| Behavior | Response configuration |
