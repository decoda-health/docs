# Rules Page - User Actions Guide

## Overview

The Rules page manages rule sets for patient collections and billing. Rule sets define how patient balances are processed, aging rules, and automated collection workflows.

---

## Available Actions

### 1. View All Rule Sets

**Purpose:** Browse existing rule sets.

**Step-by-Step Instructions:**

1. **Navigate to Rules**
   - Settings → Rules

2. **View rule set list**

3. **Click to select**

---

### 2. Create Rule Set

**Purpose:** Add new collection rules.

**Step-by-Step Instructions:**

1. **Click "Create Rule Set"**

2. **Enter rule set name**

3. **Configure rules:**
   - Age ranges
   - Actions per range
   - Notification settings

4. **Save rule set**

---

### 3. Edit Rule Set

**Purpose:** Modify existing rules.

**Step-by-Step Instructions:**

1. **Select rule set**

2. **Modify configuration**

3. **Save changes**

---

### 4. Delete Rule Set

**Purpose:** Remove a rule set.

**Step-by-Step Instructions:**

1. **Select rule set**

2. **Click delete**

3. **Confirm deletion**

---

## Rule Set Components

| Component | Purpose |
|-----------|---------|
| Age Ranges | Days since balance created |
| Actions | What happens at each stage |
| Notifications | SMS/Email triggers |

---

## Usage

Rule sets used for:
- Patient uploads (collections)
- Automated billing workflows
- Balance aging reports
