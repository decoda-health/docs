# Calendar Appearance Page - User Actions Guide

## Overview

The Calendar Appearance page customizes how appointments display on the calendar. Configure status colors/styles and appointment icons to quickly identify appointment types and states.

---

## Available Actions

### 1. Configure Status Styles

**Purpose:** Customize appointment appearance by status.

**Step-by-Step Instructions:**

1. **Navigate to Calendar Appearance**
   - Settings → Calendar Appearance

2. **View Status Styles section**

3. **For each status:**
   - Set background color
   - Set text color
   - Set border style

4. **Changes save automatically**

---

### 2. Add Appointment Icon

**Purpose:** Create visual indicator for appointment types.

**Step-by-Step Instructions:**

1. **Find Appointment Icons section**

2. **Click "Add Icon" button**

3. **Select icon type:**
   - Booking method icons
   - Checklist completion icons

4. **Choose icon image**

5. **Configure display settings**

6. **Save**

---

### 3. Edit Existing Icon

**Purpose:** Modify icon configuration.

**Step-by-Step Instructions:**

1. **Find icon in list**

2. **Click edit button**

3. **Modify settings**

4. **Save changes**

---

### 4. Delete Icon

**Purpose:** Remove appointment icon.

**Step-by-Step Instructions:**

1. **Find icon to remove**

2. **Click delete button**

3. **Confirm deletion**

---

## Icon Types

| Type | Purpose |
|------|---------|
| Self-Booked | Patient scheduled online |
| Staff-Booked | Staff scheduled |
| Rescheduled | Appointment moved |
| Forms Complete | All forms submitted |
| Check-In Complete | Patient checked in |
