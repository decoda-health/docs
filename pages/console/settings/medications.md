# Medications Page - User Actions Guide

## Overview

The Medications page manages the medication inventory for the organization. Staff can add, edit, and archive medications with pricing, unit types, categories, and template settings.

---

## Available Actions

### 1. View All Medications

**Purpose:** Browse medication inventory.

**Step-by-Step Instructions:**

1. **Navigate to Medications**
   - Settings → Medications

2. **View medication table:**
   - Name and ID
   - Brand
   - Unit
   - Price
   - Category
   - Template status

---

### 2. Add New Medication

**Purpose:** Create new medication entry.

**Step-by-Step Instructions:**

1. **Click "Add New Medication"**
   - Top right

2. **Drawer opens**

3. **Enter details:**
   - Name
   - Brand name (optional)
   - Unit type
   - Price
   - Category

4. **Save medication**

---

### 3. Edit Medication

**Purpose:** Modify medication details.

**Step-by-Step Instructions:**

1. **Click on medication row**
   - Opens drawer

2. **Modify fields**

3. **Save changes**

---

### 4. Archive Medication

**Purpose:** Remove medication from active list.

**Step-by-Step Instructions:**

1. **Click menu (three dots)**

2. **Select "Archive Medication"**

3. **Confirm archival**

4. **Medication hidden**
   - Historical data preserved

---

### 5. Filter Medications

**Purpose:** Find specific medications.

**Step-by-Step Instructions:**

1. **View filter bar**

2. **Filter by:**
   - Name (text search)
   - Brand (multiselect)
   - Category (multiselect)

3. **Results update**

---

### 6. Sort Medications

**Purpose:** Organize medication list.

**Step-by-Step Instructions:**

1. **Click column header**
   - Name, Brand, Price, Category

2. **Toggle sort direction**
   - Up arrow = ascending
   - Down arrow = descending

---

### 7. Clear Filters

**Purpose:** Reset to show all.

**Step-by-Step Instructions:**

1. **When filters active**

2. **Click "Clear Filters"**

3. **All medications shown**

---

## Table Columns

| Column | Description |
|--------|-------------|
| Medication | Name and ID |
| Brand | Brand name if set |
| Unit | Measurement unit |
| Price | Per-unit cost |
| Category | Grouping category |
| Template | Template indicator |

---

## Unit Types

Available units:
- ml (milliliters)
- mg (milligrams)
- units
- tablets
- syringes

---

## Template Medications

Medications marked as template:
- Show green "Template" badge
- Used as defaults for new records
