# Create Cart

The Create Cart page allows staff to build a shopping cart with products and services for a patient before checkout.

## Create a Cart

<Steps>
  <Step title="Navigate from Carts page">
    Click **Create Cart** button
  </Step>
  <Step title="Select patient">
    Search and select the patient
  </Step>
  <Step title="Add items">
    Browse and add products/services to the cart
  </Step>
  <Step title="Adjust quantities">
    Use +/- buttons for each item
  </Step>
  <Step title="Apply discounts">
    Add any applicable discounts
  </Step>
  <Step title="Save cart">
    Cart is created in Draft status
  </Step>
</Steps>

## Cart Options

After saving, the cart can be:
- **Converted to charge** - For immediate payment
- **Sent as quote** - For patient review and acceptance
- **Edited** - To add or remove items
- **Duplicated** - To create a copy

<Tip>
Carts can also be created during checkout. Use this page when you want to save items for later.
</Tip>

## Troubleshooting

| Issue | Solution |
|-------|----------|
| Can't find patient | Create new patient or check spelling |
| Item not available | Verify product/service is active |
| Save failed | Check all required fields |
