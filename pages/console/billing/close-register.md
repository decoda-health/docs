# Close Register Page - User Actions Guide

## Overview

The Close Register page allows staff to count cash at the end of a shift or day and reconcile it with the expected balance. It provides a detailed cash counting interface for bills and coins, calculates discrepancies, and records the closing balance.

---

## Available Actions

### 1. Select Location (Multi-Location)

**Purpose:** Choose which location to close register for.

**Step-by-Step Instructions:**

1. **If multiple locations exist:**
   - Location dropdown appears

2. **Click dropdown**
   - Shows available locations

3. **Select location**
   - Expected balance loads for that location

---

### 2. Count Bills

**Purpose:** Enter count of each bill denomination.

**Step-by-Step Instructions:**

1. **Locate "Bills" section**
   - Left side of counting grid

2. **For each denomination:**
   - **$100 bills:** Enter count
   - **$50 bills:** Enter count
   - **$20 bills:** Enter count
   - **$10 bills:** Enter count
   - **$5 bills:** Enter count
   - **$1 bills:** Enter count

3. **View totals**
   - Each field shows calculated amount
   - Bills Total shows sum of all bills

---

### 3. Count Coins

**Purpose:** Enter count of each coin denomination.

**Step-by-Step Instructions:**

1. **Locate "Coins" section**
   - Right side of counting grid

2. **For each denomination:**
   - **Quarters (25¢):** Enter count
   - **Dimes (10¢):** Enter count
   - **Nickels (5¢):** Enter count
   - **Pennies (1¢):** Enter count

3. **View totals**
   - Each field shows calculated amount
   - Coins Total shows sum of all coins

---

### 4. View Expected Balance

**Purpose:** See what the cash balance should be.

**Step-by-Step Instructions:**

1. **Locate "Expected Cash Balance" section**
   - Left column

2. **Review breakdown:**
   - **Last closing balance:** Previous close amount
   - **Cash payments since close:** Payments received
   - **Cash refunds since close:** Refunds given (if any)
   - **Adjustments since close:** Pay-ins/pay-outs
   - **Expected Total:** Calculated expected cash

3. **Note last close date**
   - Shows when register was last closed

---

### 5. Review Reconciliation Summary

**Purpose:** Compare counted cash to expected.

**Step-by-Step Instructions:**

1. **Locate "Reconciliation Summary" section**
   - Below expected balance

2. **Review:**
   - **Counted Total:** Sum of your count
   - **Expected Total:** What should be there
   - **Discrepancy:** Difference

3. **Discrepancy shows:**
   - Green if over (positive)
   - Red if short (negative)
   - Gray if exact match

---

### 6. Reset Count

**Purpose:** Clear all entered counts and start over.

**Step-by-Step Instructions:**

1. **Click "Reset Count" button**
   - Above counting grid

2. **All fields reset to zero**
   - Counted total becomes $0.00

---

### 7. Close Register

**Purpose:** Submit the closing balance.

**Step-by-Step Instructions:**

1. **Verify location is selected**

2. **Verify cash count is complete**

3. **Review discrepancy**
   - Note if over or short

4. **Click "Close Register" button**
   - Top right corner

5. **Wait for confirmation**
   - Success message appears
   - Redirected to Cash Balances page

---

### 8. Cancel Closing

**Purpose:** Return without closing register.

**Step-by-Step Instructions:**

1. **Click "Cancel" button**
   - Or back arrow

2. **Return to Cash Balances page**
   - No changes saved

## Discrepancy Interpretation

| Discrepancy | Meaning | Color |
|-------------|---------|-------|
| +$X.XX (Over) | More cash than expected | Green |
| -$X.XX (Short) | Less cash than expected | Red |
| $0.00 | Cash matches exactly | Gray |

## Tips for Accurate Counting

1. **Count multiple times** - Verify your count
2. **Separate denominations** - Organize before counting
3. **Use calculator** - Double-check totals
4. **Count coins last** - Bills are easier to count first
5. **Quiet environment** - Minimize distractions

## Troubleshooting

| Issue | Solution |
|-------|----------|
| Location not loading | Select from dropdown |
| Expected balance empty | Wait for data to load |
| Submit button disabled | Select location first |
| Counts not calculating | Check for non-numeric input |
| Large discrepancy | Recount cash, check for errors |
