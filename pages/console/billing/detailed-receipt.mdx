# Detailed Receipt

The Detailed Receipt page displays a comprehensive receipt for a specific charge, including all payment history and itemized details.

## View Receipt

<Steps>
  <Step title="Access from Charges">
    Click actions menu (⋮) → **Detailed Receipt**
  </Step>
  <Step title="Or from Payments">
    Click **Generate Detailed Receipt**
  </Step>
  <Step title="Receipt opens">
    Full details displayed in new tab
  </Step>
</Steps>

## Receipt Contents

The detailed receipt includes:
- **Practice Information**: Name, address, contact
- **Patient Information**: Name, ID, contact details
- **Charge Details**: Date, ID, service date
- **Line Items**: All services/products with quantities and prices
- **Discounts**: Any discounts applied
- **Payments**: All payments made toward the charge
- **Balance**: Outstanding amount if any

## Actions

| Action | Description |
|--------|-------------|
| **Print** | Open print dialog |
| **Download PDF** | Save as PDF file |
| **Send to Patient** | Email or text receipt link |

<Tip>
This page is designed to be print-friendly with practice branding and professional formatting.
</Tip>

## Troubleshooting

| Issue | Solution |
|-------|----------|
| Receipt not loading | Check charge ID is valid |
| Missing payments | Ensure payments are linked to charge |
| Print layout wrong | Use browser print preview to adjust |
