# Detailed Receipt Page - User Actions Guide

## Overview

The Detailed Receipt page generates comprehensive receipts for insurance reimbursement or tax purposes. It includes practice information, patient details, itemized services, medical codes (ICD-10 and CPT), and payment history. The page is optimized for printing.

---

## Available Actions

### 1. View Detailed Receipt

**Purpose:** See complete charge and payment details.

**Step-by-Step Instructions:**

1. **Navigate to detailed receipt**
   - From charge details → "Detailed Receipt"
   - Or direct URL with charge ID

2. **View receipt sections:**
   - Practice header (logo, name, address)
   - Receipt title and charge number
   - Patient information
   - Services table
   - Medical codes
   - Payment summary

---

### 2. Print Receipt

**Purpose:** Create physical copy for patient.

**Step-by-Step Instructions:**

1. **Click "Print Receipt" button**
   - Top right corner
   - Or press Ctrl/Cmd + P

2. **Configure print settings:**
   - Select printer
   - Paper size: Letter
   - Enable background graphics

3. **Print**
   - Optimized print layout applies
   - Clean professional output

---

### 3. Add ICD-10 Diagnosis Codes

**Purpose:** Include diagnosis codes for insurance.

**Step-by-Step Instructions:**

1. **Find "ICD-10 Diagnosis Codes" section**
   - In services table

2. **Click edit icon**
   - Enables edit mode

3. **Search for code:**
   - Type code or description
   - E.g., "E66.9" or "Obesity"

4. **Select from results**
   - Click to add code
   - Code and description shown

5. **Add more codes if needed**
   - Click + to add another row

---

### 4. Add CPT Procedure Codes

**Purpose:** Include procedure codes for billing.

**Step-by-Step Instructions:**

1. **Find "CPT Procedure Codes" section**

2. **Click edit icon**

3. **Search for code:**
   - Type code or description
   - E.g., "99213" or "office visit"

4. **Select from results**
   - Click to add code

5. **Add additional codes as needed**

---

### 5. Remove Medical Code

**Purpose:** Delete an added code.

**Step-by-Step Instructions:**

1. **Find code to remove**

2. **Click edit icon to enter edit mode**

3. **Clear the code field**
   - Or click remove button

4. **Code is removed**

---

### 6. Add Custom Description

**Purpose:** Include additional notes or identifiers.

**Step-by-Step Instructions:**

1. **Find "Custom Description" section**

2. **Click edit icon**

3. **Enter text:**
   - TIN numbers
   - Special instructions
   - Additional notes

4. **Click out to save**
   - Description appears on receipt

---

### 7. Save as PDF

**Purpose:** Create digital copy.

**Step-by-Step Instructions:**

1. **Click "Print Receipt" button**
   - Or Ctrl/Cmd + P

2. **Change destination:**
   - "Save as PDF"
   - "Microsoft Print to PDF"

3. **Choose save location**

4. **Click "Save"**
   - PDF created

---

## Receipt Sections

### Header
- Practice logo
- Practice/location name
- Full address
- Phone number

### Receipt Information
- "Detailed Receipt" title
- Charge ID number
- Date generated

### Patient Information
- Patient name
- Patient ID
- Phone number
- Email (if available)
- Full address

### Services Table
- Date of service
- Charge items with quantities
- Individual prices
- Discounts applied
- Medical codes section
- Total amount

### Payment Summary
- Each payment made
- Payment date
- Payment method (card type, last 4 digits)
- Amount per payment
- Total paid

## Medical Code Search

### ICD-10 Codes
- Search by code (e.g., "E66.9")
- Search by description (e.g., "obesity")
- Shows short and long descriptions
- Auto-populates description

### CPT Codes
- Search by code (e.g., "99213")
- Search by procedure (e.g., "office visit")
- Shows procedure description
- Used for billing procedures
