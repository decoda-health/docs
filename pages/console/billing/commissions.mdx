# Commissions

The Commissions page displays provider commission calculations and earnings. View commission breakdowns, filter by date range, and export reports.

## View Commissions

<Steps>
  <Step title="Navigate to Commissions">
    Go to **Billing** → **Commissions**
  </Step>
  <Step title="Select date range">
    Choose the period to view
  </Step>
  <Step title="Review breakdown">
    See earnings by provider, service, and date
  </Step>
</Steps>

## Commission Details

<CardGroup cols={2}>
  <Card title="Provider Breakdown" icon="user-doctor">
    View earnings for each provider
  </Card>
  <Card title="Service Breakdown" icon="clipboard-list">
    See commission by service type
  </Card>
</CardGroup>

## Filter Commissions

| Filter | Description |
|--------|-------------|
| **Date Range** | Start and end dates |
| **Provider** | Specific provider |
| **Location** | By practice location |

## Export Report

<Steps>
  <Step title="Configure filters">
    Set the date range and other filters
  </Step>
  <Step title="Click Export">
    Download as CSV or Excel
  </Step>
</Steps>

## Edit Commission Rules

Click **Edit** to access commission structure configuration. See Settings → Commission Structures for detailed setup.

## Troubleshooting

| Issue | Solution |
|-------|----------|
| Provider missing | Check provider has commission structure assigned |
| Amounts incorrect | Verify commission rules configuration |
| Export empty | Adjust date range filters |
