# Checkout Page - User Actions Guide

## Overview

The Checkout page is the primary payment collection interface. It guides staff through creating charges, adding items, and processing payments with support for multiple payment methods including cards, cash, POS terminals, and payment plans.

---

## Available Actions

### 1. Select Patient

**Purpose:** Choose which patient to charge.

**Step-by-Step Instructions:**

1. **Locate patient selector**
   - In charge edit step

2. **Search for patient**
   - Type name or phone
   - Select from results

3. **Or create new patient**
   - Click "New Patient"
   - Enter required info

---

### 2. Add Services/Products

**Purpose:** Add items to the charge.

**Step-by-Step Instructions:**

1. **Browse available items**
   - Services list
   - Products list

2. **Click to add item**
   - Item added to cart

3. **Adjust quantity**
   - Use +/- buttons
   - Or type quantity

4. **View running total**
   - Updates automatically

---

### 3. Apply Discount

**Purpose:** Reduce the charge total.

**Step-by-Step Instructions:**

1. **Click "Add Discount"**
   - In cart section

2. **Choose discount type:**
   - Percentage off
   - Fixed amount off

3. **Enter discount value**
   - Amount or percentage

4. **Apply discount**
   - Total updates

---

### 4. Apply Package Credits

**Purpose:** Use pre-purchased package services.

**Step-by-Step Instructions:**

1. **View available packages**
   - Patient's packages shown

2. **Select package to apply**
   - Click on package

3. **Apply to items**
   - Matching items reduced
   - Package balance decreases

---

### 5. Apply Membership Benefits

**Purpose:** Use membership discounts.

**Step-by-Step Instructions:**

1. **View membership status**
   - Active memberships shown

2. **Benefits auto-applied**
   - Member pricing shown
   - Discounts calculated

---

### 6. Proceed to Payment

**Purpose:** Move to payment collection.

**Step-by-Step Instructions:**

1. **Review cart**
   - Verify items and total

2. **Click "Continue to Payment"**
   - Moves to payment step

---

### 7. Pay with Card on File

**Purpose:** Use saved payment method.

**Step-by-Step Instructions:**

1. **View saved cards**
   - Patient's cards displayed

2. **Select card**
   - Click to choose

3. **Process payment**
   - Click "Charge $X.XX"
   - Payment processes

---

### 8. Pay with New Card

**Purpose:** Enter new card details.

**Step-by-Step Instructions:**

1. **Enter card details directly in form:**
   - Card number
   - Expiration (MM/YY)
   - CVV
   - ZIP code

2. **Save card (optional)**
   - Toggle to save for future

3. **Process payment**
   - Click "Charge $X.XX"

---

### 9. Pay with POS Terminal

**Purpose:** Process in-person card payment.

**Step-by-Step Instructions:**

1. **Select POS Terminal**
   - Choose terminal device

2. **Initiate payment**
   - Click "Charge $X.XX on POS"

3. **Wait for customer action**
   - Page shows processing
   - Customer interacts with terminal

4. **Complete transaction**
   - Insert/tap/swipe card
   - Enter PIN if required
   - Approve amount

5. **View result**
   - Success or failure shown

---

### 10. Pay with Cash

**Purpose:** Record cash payment.

**Step-by-Step Instructions:**

1. **Select "Cash" payment**

2. **Enter amount tendered**
   - What customer gave

3. **View change due**
   - Calculated automatically

4. **Complete payment**
   - Record transaction

---

### 11. Add Tip

**Purpose:** Include gratuity.

**Step-by-Step Instructions:**

1. **Locate tip section**
   - In payment step

2. **Enter tip amount**
   - Fixed amount or percentage
   - Select preset or custom

3. **Apply tip**
   - Added to total

---

### 12. Set Up Payment Plan

**Purpose:** Split payment into installments.

**Step-by-Step Instructions:**

1. **Click "Setup Payment Plan"**
   - Plan options appear

2. **Configure plan:**
   - Down payment amount
   - Number of installments
   - Payment frequency

3. **Enter card for autopay**
   - Card used for scheduled payments

4. **Confirm plan**
   - Plan created
   - First payment processed

---

### 13. Preview Charge

**Purpose:** See charge details before finalizing.

**Step-by-Step Instructions:**

1. **Click "Preview" button**
   - In header

2. **Review all details:**
   - Items and prices
   - Discounts applied
   - Final total

3. **Close preview**
   - Return to editing

---

### 14. Cancel Checkout

**Purpose:** Abandon the checkout process.

**Step-by-Step Instructions:**

1. **Click back/cancel**
   - In header

2. **Confirm cancellation**
   - Warning may appear
   - Unsaved changes lost

---

### 15. Print Receipt

**Purpose:** Generate receipt after payment.

**Step-by-Step Instructions:**

1. **Complete payment**
   - Success screen shown

2. **Click "Print Receipt"**
   - Opens print dialog

3. **Print or save PDF**
   - Choose printer
   - Or save as PDF

---

### 16. Send Receipt

**Purpose:** Email/text receipt to patient.

**Step-by-Step Instructions:**

1. **On success screen**

2. **Click "Send Receipt"**
   - Options appear

3. **Choose method:**
   - Email receipt
   - Text receipt link

4. **Confirm send**
   - Receipt delivered

---

## Permissions

All checkout actions are available to users with appropriate billing access. Specific permissions may be configured per role in Settings.

---

## Troubleshooting

| Issue | Solution |
|-------|----------|
| Card declined | Try different card, check details |
| POS not connecting | Check terminal status, restart |
| Can't apply package | Verify package has remaining balance |
| Discount not applying | Check discount rules |
| Payment stuck processing | Wait, then refresh if needed |
