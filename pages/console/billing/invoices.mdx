# Invoices

The Invoices page manages outstanding debt collection and patient outreach for unpaid balances. View, filter, and manage invoices, create new ones, and track payment status.

## Invoice Actions

Click on an invoice row to view details, then choose from available actions:

| Action | Description |
|--------|-------------|
| **Send Reminder** | Send payment reminder to patient |
| **Mark as Paid** | Manually mark invoice as paid |
| **Void Invoice** | Cancel the invoice |
| **Edit Details** | Modify invoice information |
| **Download PDF** | Download printable invoice |

## Invoice Table Columns

| Column | Description |
|--------|-------------|
| Patient | Patient name and ID |
| Amount | Total invoice amount |
| Status | Current payment status |
| Due Date | When payment is due |
| Created | Invoice creation date |
| Actions | Available actions menu |

## Invoice Statuses

| Status | Meaning |
|--------|---------|
| **Pending** | Awaiting payment |
| **Paid** | Fully paid |
| **Partial** | Partially paid |
| **Overdue** | Past due date |
| **Voided** | Cancelled invoice |

<Warning>
Overdue invoices may trigger automated reminder workflows if configured in settings.
</Warning>

## Troubleshooting

| Issue | Solution |
|-------|----------|
| Invoice not showing | Check filters applied |
| Can't create invoice | Verify patient has outstanding charges |
| Send reminder fails | Check patient contact info |
| Filters not working | Clear and reapply |
