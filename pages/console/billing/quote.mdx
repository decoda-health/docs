# Quotes

The Quote page displays treatment quotes that have been sent to patients for review and acceptance.

## View Quote

<Steps>
  <Step title="Access quote">
    Navigate from Carts page or direct link
  </Step>
  <Step title="Review details">
    See all items, prices, and terms
  </Step>
  <Step title="Check status">
    View if quote is pending, accepted, or rejected
  </Step>
</Steps>

## Quote Statuses

| Status | Description |
|--------|-------------|
| **Pending** | Awaiting patient response |
| **Accepted** | Patient approved the quote |
| **Rejected** | Patient declined |
| **Expired** | Quote passed expiration date |
| **Converted** | Quote converted to charge |

## Quote Actions

| Action | Description |
|--------|-------------|
| **Resend** | Send quote again to patient |
| **Edit** | Modify quote details (if still pending) |
| **Convert to Charge** | Create charge from accepted quote |
| **Cancel** | Cancel the quote |

## Send Quote

<Steps>
  <Step title="Create cart">
    Add items to a cart
  </Step>
  <Step title="Select Create Quote">
    From cart actions menu
  </Step>
  <Step title="Set expiration">
    Choose when quote expires
  </Step>
  <Step title="Add terms">
    Include any terms or notes
  </Step>
  <Step title="Send">
    Patient receives quote via SMS/email
  </Step>
</Steps>

<Info>
Patients can view and respond to quotes from the link sent to them.
</Info>

## Troubleshooting

| Issue | Solution |
|-------|----------|
| Quote not delivered | Check patient contact info |
| Can't convert to charge | Quote must be accepted first |
| Quote expired | Create a new quote |
