# Check-In Dashboard

The Dashboard provides real-time patient check-in status and flow tracking. View today's appointments, check in patients, and monitor wait times.

    Locate the patient's appointment card
  </Step>
  <Step title="Click Check In">
    Button on the appointment card
  </Step>
  <Step title="Confirm check-in">

  </Step>

## Update Status

| Status | Action |
|--------|--------|
| **Check In** | Mark patient as arrived |
| **In Room** | Move patient to appointment |
| **Complete** | Finish the appointment |
| **No Show** | Mark patient as no-show |

## View Wait Times

The dashboard displays:
- Average wait time for the day
- Current wait time per patient
- Color coding for long waits

<Warning>
Patients waiting longer than the threshold may trigger alerts.
</Warning>

## Troubleshooting

| Issue | Solution |
|-------|----------|
| Appointments not showing | Check date and location filters |
| Can't check in | Verify appointment is for today |
| Status won't update | Check permissions |
