# Clock-In History

Track staff time entries for payroll and attendance. View historical clock-in/out records, filter by staff member, and export for payroll.

<Info>
This feature requires the CLOCK_IN module to be enabled for your tenant.
</Info>

## View Time Entries

<Steps>
  <Step title="Navigate to Clock-In History">
    Go to **Check-In** â†’ **Clock-In History**
  </Step>
  <Step title="Select date range">
    Choose the period to view
  </Step>
  <Step title="Review entries">
    See clock-in, clock-out times, and total hours
  </Step>
</Steps>

## Filter Entries

| Filter | Description |
|--------|-------------|
| **Date Range** | Select start and end dates |
| **Staff Member** | Filter by specific user |
| **Location** | Filter by work location |

## Time Entry Details

Each entry shows:
- Staff member name
- Clock-in time
- Clock-out time
- Total hours worked
- Location
- Notes (if any)

## Export for Payroll

<Steps>
  <Step title="Set filters">
    Select date range and staff
  </Step>

  </Step>

  </Step>
