# Forms List Page - User Actions Guide

## Overview

The Forms List page is the management hub for all patient intake and consent forms. Staff can view, create, edit, send, and manage form submissions from this central location.

---

## Available Actions

### 1. View All Forms

**Purpose:** See all forms configured for the practice.

**Step-by-Step Instructions:**

1. **Navigate to Forms List**
   - Click "Check-In" in navigation
   - Click "Forms" submenu

2. **View form cards/list**
   - Each form displays:
     - Form name
     - Description
     - Status (active/archived)
     - Submission count

---

### 2. Search Forms

**Purpose:** Find a specific form quickly.

**Step-by-Step Instructions:**

1. **Locate search box**
   - At top of forms list
   - Placeholder: "Search forms..."

2. **Enter search term**
   - Type form name
   - Results filter as you type

3. **View results**
   - Matching forms shown
   - Non-matching hidden

---

### 3. Create New Form

**Purpose:** Build a new intake or consent form.

**Step-by-Step Instructions:**

1. **Click "Create Form" button**
   - Located in header area
   - Primary action button

2. **Choose form type**
   - Consent form
   - Intake form
   - Custom form

3. **Navigate to form builder**
   - Redirects to create-consent page
   - Begin building form

---

### 4. Edit Existing Form

**Purpose:** Modify an existing form's structure or content.

**Step-by-Step Instructions:**

1. **Locate the form to edit**
   - Find in list or search

2. **Click edit action**
   - Pencil icon or "Edit" button
   - Or click form name

3. **Make changes in form builder**
   - Modify questions
   - Update settings
   - Save changes

---

### 5. View Form Submissions

**Purpose:** See all submissions for a specific form.

**Step-by-Step Instructions:**

1. **Locate the form**
   - Find in list

2. **Click submissions count/button**
   - Shows number of submissions
   - Opens submissions drawer

3. **Browse submissions**
   - List of patient submissions
   - Click to view details

4. **View individual submission**
   - See all answers
   - View signature (if applicable)
   - Download PDF

---

### 6. Send Form to Patient

**Purpose:** Send a form link to a patient.

**Step-by-Step Instructions:**

1. **Select the form**
   - Click on form card
   - Or use form menu

2. **Click "Send" or share icon**
   - Send options appear

3. **Choose delivery method:**

   **Via SMS:**
   - Enter patient phone number
   - Customize message (optional)
   - Click Send

   **Via Email:**
   - Enter patient email
   - Customize subject/message
   - Click Send

4. **Confirm sent**
   - Success message appears
   - Patient receives link

---

### 7. Generate Form QR Code

**Purpose:** Create a QR code patients can scan to access the form.

**Step-by-Step Instructions:**

1. **Select the form**
   - Find in list

2. **Click QR code option**
   - QR icon or menu option

3. **View QR code**
   - Code displays in modal
   - Ready to scan

4. **Download or print**
   - Save QR code image
   - Print for display in office

---

### 8. Bulk Select Forms

**Purpose:** Select multiple forms for bulk actions.

**Step-by-Step Instructions:**

1. **Enable selection mode**
   - Click checkbox on first form
   - Or click "Select" button

2. **Select multiple forms**
   - Click checkboxes on desired forms
   - "Select All" for all forms

3. **Perform bulk action**
   - Send selected forms to patient
   - Export selected forms
   - Archive selected forms

---

### 9. Send Multiple Forms to Patient

**Purpose:** Send several forms at once to a patient.

**Step-by-Step Instructions:**

1. **Select forms to send**
   - Check multiple form checkboxes

2. **Click "Send Selected"**
   - Bulk send option appears

3. **Enter patient information**
   - Phone number and/or email

4. **Send forms**
   - All selected forms sent
   - Patient receives consolidated link

---

### 10. Archive a Form

**Purpose:** Deactivate a form while preserving submissions.

**Step-by-Step Instructions:**

1. **Locate the form**
   - Find in list

2. **Click archive option**
   - In form menu or actions

3. **Confirm archival**
   - Warning if form has submissions
   - Click "Archive" to confirm

4. **Form moves to archived**
   - No longer available for sending
   - Existing submissions preserved

---

### 11. Edit Default Message

**Purpose:** Customize the default message sent with a form.

**Step-by-Step Instructions:**

1. **Access form settings**
   - Click settings icon on form
   - Or edit form

2. **Find message settings**
   - "Default Message" section

3. **Customize message**
   - Edit SMS message text
   - Edit email subject/body

4. **Save changes**
   - New message used for future sends

---

### 12. Export Form as PDF

**Purpose:** Download a blank form template.

**Step-by-Step Instructions:**

1. **Select the form**
   - Click on form

2. **Click export/download**
   - PDF icon or menu option

3. **Download PDF**
   - Blank form downloads
   - Suitable for printing

---

## Form Status Indicators

| Status | Meaning |
|--------|---------|
| Active | Form is live and can be sent |
| Archived | Form is deactivated |
| Draft | Form is being created |

## Troubleshooting

| Issue | Solution |
|-------|----------|
| Form not sending | Check phone/email is valid |
| QR code not scanning | Increase code size |
| Can't edit form | Check permissions |
| Submissions missing | Check date filters |
| Archived form needed | Unarchive the form |
