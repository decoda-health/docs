# Calendar Page - User Actions Guide

## Overview

The Calendar page is the primary appointment management interface for healthcare providers and staff. It displays appointments in a visual calendar format with filtering, navigation, and appointment management capabilities.

---

## Available Actions

### 1. Toggle Calendar Sidebar

**Purpose:** Show or hide the left sidebar containing filters and mini-calendar.

**Step-by-Step Instructions:**

1. **Locate the sidebar toggle button**
   - Small arrow button on the right edge of the sidebar
   - Only visible on desktop/tablet devices
   - Shows left arrow (◀) when sidebar is open
   - Shows right arrow (▶) when sidebar is collapsed

2. **Click to collapse the sidebar**
   - Click the toggle button when sidebar is open
   - Sidebar slides out of view to the left
   - Calendar expands to use full width
   - More appointments visible at once

3. **Click to expand the sidebar**
   - Click the toggle button when sidebar is collapsed
   - Sidebar slides back into view
   - Calendar contracts to original width
   - Filters and mini-calendar become accessible

**Mobile Behavior:**
- Sidebar appears as an overlay
- Click outside the sidebar to close it
- Backdrop blur indicates overlay mode

---

### 2. Navigate to Previous/Next Time Periods

**Purpose:** View appointments for different days, weeks, or months.

**Step-by-Step Instructions:**

1. **Locate navigation controls**
   - Arrow buttons near the date display
   - "Today" button to return to current date

2. **Navigate to previous period**
   - Click the left arrow (◀ or <)
   - View shifts based on current view:
     - Day view: Previous day
     - Week view: Previous week
     - Month view: Previous month

3. **Navigate to next period**
   - Click the right arrow (▶ or >)
   - View shifts based on current view

4. **Return to today**
   - Click the "Today" button
   - Calendar jumps to current date
   - Current time indicator shown

---

### 3. Switch Calendar Views

**Purpose:** Change between day, week, and month views.

**Step-by-Step Instructions:**

1. **Locate view selector**
   - Buttons or dropdown near navigation controls
   - Options: Day, Week, Month (varies by configuration)

2. **Select Day View**
   - Click "Day" button
   - Shows single day with detailed time slots
   - Best for managing individual day's schedule

3. **Select Week View**
   - Click "Week" button
   - Shows 5 or 7 days (based on weekend settings)
   - Good for weekly planning overview

4. **Select Month View**
   - Click "Month" button (if available)
   - Shows entire month grid
   - Appointments shown as compact blocks

---

### 4. Filter Appointments by Provider

**Purpose:** Show only appointments for specific providers.

**Step-by-Step Instructions:**

1. **Locate provider filter**
   - In the sidebar (if expanded)
   - Or in the filter dropdown

2. **Open provider selector**
   - Click the provider dropdown or checkboxes
   - List of all active providers appears

3. **Select providers to display**
   - Check/click providers you want to see
   - Uncheck providers to hide their appointments
   - "Select All" / "Clear All" for bulk actions

4. **View filtered results**
   - Calendar immediately updates
   - Only selected providers' appointments shown
   - Color coding per provider maintained

---

### 5. Filter Appointments by Location

**Purpose:** Show only appointments for specific locations.

**Step-by-Step Instructions:**

1. **Locate location filter**
   - In the sidebar or filter area
   - Location dropdown or selector

2. **Open location selector**
   - Click the location dropdown
   - All practice locations listed

3. **Select locations**
   - Check locations you want to view
   - Multi-select supported
   - Changes apply immediately

4. **Clear location filter**
   - Uncheck all or click "Clear"
   - Shows all locations again

---

### 6. View Appointment Details

**Purpose:** See full details of a specific appointment.

**Step-by-Step Instructions:**

1. **Locate an appointment on the calendar**
   - Appointments appear as colored blocks
   - Show patient name, time, and service

2. **Click on the appointment**
   - Appointment becomes highlighted/selected
   - Details panel or modal opens

3. **Review appointment information**
   - Patient name and contact info
   - Service type and duration
   - Provider assigned
   - Appointment status
   - Notes and special instructions

4. **Close the details view**
   - Click the X button
   - Click outside the panel
   - Press Escape key

---

### 7. Create a New Appointment

**Purpose:** Schedule a new appointment for a patient.

**Step-by-Step Instructions:**

1. **Method 1: Click on empty time slot**
   - Find an open time slot on the calendar
   - Click on the desired time
   - New appointment form opens

2. **Method 2: Use create button**
   - Click "Create Event" button in the sidebar
   - New appointment form opens

3. **Fill in appointment details**
   - **Patient:** Search and select existing patient, or create new
   - **Service:** Select from available services
   - **Provider:** Choose the provider
   - **Date/Time:** Confirm or adjust the time
   - **Duration:** Usually auto-filled based on service
   - **Location:** Select if multiple locations
   - **Notes:** Add any special instructions

4. **Save the appointment**
   - Click "Save" or "Book Appointment"
   - Appointment appears on calendar
   - Confirmation may be sent to patient

---

### 8. Drag and Drop to Reschedule

**Purpose:** Quickly move an appointment to a different time.

**Step-by-Step Instructions:**

1. **Locate the appointment to move**
   - Find it on the calendar

2. **Click and hold the appointment**
   - Cursor changes to indicate dragging
   - Appointment becomes slightly transparent

3. **Drag to new time slot**
   - Move mouse/finger to desired time
   - Visual indicator shows where it will land
   - Can move between days in week view

4. **Release to drop**
   - Let go of mouse/finger
   - Confirmation dialog may appear
   - Appointment moves to new time

5. **Confirm or cancel**
   - If prompted, confirm the change
   - Or cancel to revert

**Restrictions:**
- Cannot drag to blocked times
- Cannot overlap with other appointments (unless allowed)
- May require appropriate permissions

---

### 9. Resize Appointment Duration

**Purpose:** Extend or shorten an appointment's duration.

**Step-by-Step Instructions:**

1. **Hover over appointment edge**
   - Bottom edge of the appointment block
   - Cursor changes to resize indicator

2. **Click and drag the edge**
   - Pull down to extend duration
   - Pull up to shorten duration

3. **Release to apply**
   - New duration is saved
   - Appointment block resizes

---

### 10. Change Appointment Status

**Purpose:** Update the status of an appointment (e.g., check-in, complete).

**Step-by-Step Instructions:**

1. **Click on the appointment**
   - Opens appointment details

2. **Locate status options**
   - Status dropdown or buttons
   - Common statuses:
     - Scheduled
     - Confirmed
     - Checked In
     - In Progress
     - Completed
     - No Show
     - Cancelled

3. **Select new status**
   - Click the desired status
   - Color coding may change on calendar
   - Timestamp recorded

---

### 11. Cancel an Appointment

**Purpose:** Cancel a scheduled appointment.

**Step-by-Step Instructions:**

1. **Click on the appointment**
   - Opens appointment details

2. **Locate cancel option**
   - "Cancel" button or menu option
   - May require additional click to confirm

3. **Confirm cancellation**
   - Confirmation dialog appears
   - Select cancellation reason (if required)
   - Click "Confirm Cancel"

4. **Handle cancellation**
   - Appointment marked as cancelled
   - May appear with strikethrough
   - Notification sent to patient (if configured)

---

### 12. Use Mini-Calendar Navigation

**Purpose:** Quickly jump to specific dates using the small calendar in sidebar.

**Step-by-Step Instructions:**

1. **Locate the mini-calendar**
   - Small month calendar in the sidebar
   - Shows current month by default

2. **Navigate mini-calendar months**
   - Arrow buttons to go to previous/next months
   - Click month/year header to change

3. **Click on a date**
   - Main calendar jumps to that date
   - Date becomes highlighted in mini-calendar

4. **View date context**
   - Days with appointments may be marked
   - Today's date highlighted differently

---

### 13. View Provider Columns (Multi-Provider View)

**Purpose:** See multiple providers' schedules side by side.

**Step-by-Step Instructions:**

1. **Enable multi-provider view**
   - Select multiple providers in filter
   - Or enable "Provider Columns" view option

2. **View side-by-side schedules**
   - Each provider gets a column
   - Appointments appear in respective columns
   - Time slots aligned across columns

3. **Manage visibility**
   - Toggle individual providers on/off
   - Reorder columns (if supported)

---

### 14. View Resource/Room Scheduling

**Purpose:** See availability of rooms or equipment alongside appointments.

**Step-by-Step Instructions:**

1. **Enable resource view**
   - Toggle or filter for resources
   - Shows rooms, equipment, etc.

2. **View resource allocation**
   - Resources appear as additional columns/rows
   - Bookings show which appointments use resources

3. **Book resources with appointments**
   - When creating appointment, select resource
   - Resource shows as occupied for that time

---

## Calendar Color Coding

| Color | Meaning |
|-------|---------|
| Provider colors | Each provider has unique color |
| Gray/Strikethrough | Cancelled appointments |
| Faded/Translucent | Past appointments (if setting enabled) |
| Highlighted border | Selected appointment |
| Blocked pattern | Time blocks/unavailable |

## Mobile-Specific Features

- Swipe left/right to navigate days
- Tap and hold for context menu
- Simplified single-day view by default
- Sidebar accessible via overlay
- Touch-optimized appointment selection

---

## Accessibility Features

- Full keyboard navigation
- Screen reader announcements for appointments
- ARIA labels on all interactive elements
- Focus management in dialogs
- High contrast mode support
- Semantic time grid structure

---

## Troubleshooting

| Issue | Solution |
|-------|----------|
| Calendar not loading | Refresh page, check connection |
| Appointments not showing | Check provider/location filters |
| Can't drag appointments | Check permissions, enable drag setting |
| Wrong time zone showing | Check profile time zone settings |
| Overlapping appointments | Some practices allow this, or conflict exists |
| Slow performance | Reduce visible providers, use day view |
