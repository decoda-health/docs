# Create Form

Build custom consent and intake forms with the form builder. Add various question types, conditional logic, and signature requirements.

## Create Form

<Steps>
  <Step title="Add form title">
    Enter a descriptive name
  </Step>
  <Step title="Add description">
    Optional explanation for the form
  </Step>
  <Step title="Add question blocks">
    Build your form content
  </Step>
  <Step title="Save">
    Form is ready to use
  </Step>
</Steps>

## Conditional Logic

Show or hide questions based on previous answers:

<Steps>
  <Step title="Select question">
    Click the question to configure
  </Step>
  <Step title="Add condition">
    Choose "Show if" or "Hide if"
  </Step>
  <Step title="Set trigger">
    Select the question and answer that triggers the condition
  </Step>
</Steps>

## Form Settings

| Setting | Description |
|---------|-------------|
| **Required** | Mark questions as mandatory |
| **Default Message** | Customize the sending message |
| **Thank You Page** | Custom completion message |
| **Expiration** | Set form link expiration |

## Signature Requirements

<Steps>
  <Step title="Add signature block">
    From question types
  </Step>
  <Step title="Configure">
    Set as required, add label
  </Step>
  <Step title="Test">
    Preview to verify signature capture
  </Step>
</Steps>

<Tip>
Use the Preview button to see how the form appears to patients before publishing.
</Tip>

## Troubleshooting

| Issue | Solution |
|-------|----------|
| Can't add question | Check form is in edit mode |
| Condition not working | Verify question order |
| Signature not showing | Check browser compatibility |
