# Shift Management

Manage provider schedules and shifts. Create recurring schedules, add time blocks for breaks and meetings, and manage availability.

## Navigate Weeks

<Steps>
  <Step title="Use arrow buttons">
    Navigate to previous or next week
  </Step>
  <Step title="Jump to date">
    Click the date picker to go to a specific week
  </Step>
  <Step title="View providers">
    Each row shows a provider's weekly schedule
  </Step>
</Steps>

## Create Shift

<Steps>
  <Step title="Click on empty time slot">
    Or use the Add Shift button
  </Step>
  <Step title="Configure shift">
    - Provider
    - Start and end time
    - Location
    - Recurring options
  </Step>
  <Step title="Save">
    Shift appears on the calendar
  </Step>
</Steps>

## Add Time Block

Block time for lunch, meetings, or unavailability:

<Steps>
  <Step title="Click Add Block">
    On the provider's row
  </Step>
  <Step title="Set block details">
    - Block type (lunch, meeting, off)
    - Start and end time
    - Recurring pattern
  </Step>
  <Step title="Save">
    Block prevents appointments during that time
  </Step>
</Steps>

## Recurring Schedules

<CardGroup cols={2}>
  <Card title="Weekly Repeat" icon="rotate">
    Set shifts that repeat every week
  </Card>
  <Card title="Holiday Management" icon="calendar-xmark">
    Mark holidays and special closures
  </Card>
</CardGroup>

## Filter Options

- **Location**: Filter by practice location
- **Provider**: Show specific providers
- **Show blocks only**: Toggle to see only providers with blocks

<Tip>
Toggle "Show blocks only providers" to quickly see who has availability restrictions.
</Tip>

## Troubleshooting

| Issue | Solution |
|-------|----------|
| Can't create shift | Check provider exists at location |
| Block not showing | Verify date range is correct |
| Recurring not applying | Check start and end dates |
