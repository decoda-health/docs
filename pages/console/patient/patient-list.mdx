# Patient List

The Patient List page displays all patients with search, filtering, and management capabilities. Browse patient records, access details, and perform bulk operations.

## View Patients

<Steps>
  <Step title="Navigate to Patients">
    Click **Patients** in the main navigation
  </Step>
  <Step title="Browse patient list">
    See name, contact info, last visit, and status for each patient
  </Step>
  <Step title="Sort and paginate">
    Click column headers to sort, use pagination for large lists
  </Step>
</Steps>

## Search Patients

<Steps>
  <Step title="Use search bar">
    Enter patient name, phone, or email
  </Step>
  <Step title="View results">
    Table filters in real-time
  </Step>
</Steps>

## Filter Patients

| Filter | Description |
|--------|-------------|
| **Status** | Active, inactive, archived |
| **Last Visit** | Date range of last appointment |
| **Location** | Primary location |
| **Tags** | Patient tags |

## Create Patient

<Steps>
  <Step title="Click Add Patient">
    Button in the header
  </Step>
  <Step title="Fill required fields">
    First name, last name, phone or email
  </Step>
  <Step title="Add optional info">
    DOB, address, insurance, etc.
  </Step>
  <Step title="Save">
    Patient added to list
  </Step>
</Steps>

## Patient Actions

| Action | Description |
|--------|-------------|
| **View Details** | Open full patient profile |
| **Schedule** | Create new appointment |
| **Send Message** | Send SMS or email |
| **Edit** | Modify patient info |

## Bulk Operations

<Steps>
  <Step title="Select patients">
    Check multiple rows
  </Step>
  <Step title="Choose action">
    Send message, add tag, export
  </Step>
  <Step title="Execute">
    Action applied to all selected
  </Step>
</Steps>

## Export Patients

Click **Export** to download patient data as CSV or Excel.

## Troubleshooting

| Issue | Solution |
|-------|----------|
| Patient not found | Check spelling, try phone number |
| Can't add patient | Check required fields |
| Duplicate warning | Review existing patients |
