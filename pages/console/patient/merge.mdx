# Merge Patients

Combine duplicate patient records into a single unified profile. All data from merged records is consolidated.

<Warning>
Merging patients is irreversible. Review carefully before confirming.
</Warning>

## Start Merge

<Steps>
  <Step title="Access merge">
    From duplicate warning banner, or **Patients** â†’ **Merge**
  </Step>
  <Step title="Select patients">
    Choose the records to merge
  </Step>
  <Step title="Choose primary record">
    Select which record to keep as the base
  </Step>
</Steps>

## Review Data

Compare data from all selected records:
- Contact information
- Demographics
- Appointments
- Billing history
- Forms and documents
- Notes

## Resolve Conflicts

When records have conflicting data:

<Steps>
  <Step title="Review conflicts">
    Highlighted fields with different values
  </Step>
  <Step title="Select preferred value">
    Choose which value to keep
  </Step>
  <Step title="Continue">
    All non-conflicting data merges automatically
  </Step>
</Steps>

## Complete Merge

<Steps>
  <Step title="Review summary">
    Verify the merged record preview
  </Step>
  <Step title="Confirm">
    Click **Merge Patients**
  </Step>
  <Step title="Done">
    Secondary records archived, primary updated
  </Step>
</Steps>

## What Gets Merged

| Data Type | Handling |
|-----------|----------|
| **Appointments** | All combined into single timeline |
| **Charges/Payments** | All billing history merged |
| **Forms** | All submissions preserved |
| **Notes** | All notes combined |
| **Attachments** | All files moved to primary |

## Troubleshooting

| Issue | Solution |
|-------|----------|
| Can't find duplicate | Search by phone or email |
| Merge button disabled | Select at least 2 patients |
| Missing data after merge | All data preserved, check tabs |
