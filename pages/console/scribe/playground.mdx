# Scribe Playground

Test the AI Scribe functionality with sample recordings. Experiment with different inputs and see how notes are generated.

## Record Sample

<Steps>
  <Step title="Click Record">
    Start recording audio
  </Step>
  <Step title="Speak">
    Simulate a patient encounter
  </Step>
  <Step title="Stop">
    End the recording
  </Step>
  <Step title="Process">
    AI generates note from recording
  </Step>
</Steps>

## Upload Audio

<Steps>
  <Step title="Click Upload">
    Select an audio file
  </Step>
  <Step title="Process">
    File is transcribed and note generated
  </Step>
</Steps>

## View Results

After processing:
- Listen to recording playback
- Read full transcription
- View generated clinical note
- See note sections (chief complaint, assessment, plan)

## Customize Output

<Steps>
  <Step title="Select template">
    Choose note format/style
  </Step>
  <Step title="Set preferences">
    Configure section preferences
  </Step>
  <Step title="Regenerate">
    Apply settings and regenerate
  </Step>
</Steps>

<Info>

</Info>

## Troubleshooting

| Issue | Solution |

| Processing slow | Complex audio takes longer |
| Poor transcription | Check audio quality |
