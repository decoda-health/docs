# AI Scribe Notes

View and manage AI-generated clinical notes from patient encounters. Review, edit, and finalize transcriptions.

## View Notes

<Steps>
  <Step title="Navigate to Scribe">
    Go to **Scribe** in the main menu
  </Step>
  <Step title="Browse notes list">
    See patient, date, status, and provider for each note
  </Step>
  <Step title="Filter by status">
    View pending, reviewed, or finalized notes
  </Step>
</Steps>

## Note Status

| Status | Description |
|--------|-------------|
| **Pending** | AI-generated, awaiting review |
| **Reviewed** | Provider has reviewed |
| **Finalized** | Signed and locked |

## Review Note

<Steps>
  <Step title="Click on note">
    Opens the note editor
  </Step>
  <Step title="Review content">
    Read AI-generated transcription
  </Step>
  <Step title="Edit as needed">
    Make corrections or additions
  </Step>
  <Step title="Save or finalize">
    Save draft or finalize with signature
  </Step>
</Steps>

## Note Sections

Typical AI Scribe note includes:
- Chief complaint
- History of present illness
- Assessment
- Plan
- Additional notes

<Tip>
AI-generated notes are suggestions. Always review and verify clinical accuracy.
</Tip>

## Troubleshooting

| Issue | Solution |
|-------|----------|
| Note not appearing | Check date filter |
| Audio quality issues | Note may be incomplete |
| Can't finalize | Check all required sections |
