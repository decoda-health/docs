# Scribe Gallery

Browse recordings and transcriptions from AI Scribe sessions. Review audio, view transcripts, and manage recordings.

## View Recording

<Steps>
  <Step title="Click recording">
    Opens the detail view
  </Step>
  <Step title="Play audio">
    Listen to the original recording
  </Step>
  <Step title="View transcript">
    See the AI-generated transcription
  </Step>
</Steps>

## Recording Details

Each recording shows:
- Date and time
- Duration
- Patient (if linked)
- Provider
- Generated note status

## Actions

| Action | Description |
|--------|-------------|
| **Play** | Listen to audio |
| **View Transcript** | See text transcription |
| **Generate Note** | Create clinical note from recording |
| **Delete** | Remove recording |

<Warning>
Deleted recordings cannot be recovered.
</Warning>

## Troubleshooting

| Issue | Solution |
|-------|----------|
| Audio won't play | Check browser audio permissions |
| Transcript missing | Processing may still be in progress |
| Recording not found | Check date range filter |
