# Stock Add Page - User Actions Guide

## Overview

The Stock Add page allows creating new stock items for inventory tracking. It offers two methods: AI-assisted creation from images/descriptions, or manual entry. Staff can also use existing stock items as templates.

---

## Available Actions

### 1. Create Stock with AI

**Purpose:** Auto-generate stock from image or description.

**Step-by-Step Instructions:**

1. **Locate "Create with AI" card**
   - Left side of page

2. **Upload image or enter description**
   - Product photo
   - Or text description

3. **AI processes input**
   - Generates stock details

4. **Review AI suggestions**
   - Name, description, unit
   - Category

5. **Edit as needed**

6. **Save stock item**

---

### 2. Create Stock Manually

**Purpose:** Enter stock details by hand.

**Step-by-Step Instructions:**

1. **Locate "Create Manually" card**
   - Right side of page

2. **Click to start**
   - Opens form

3. **Or search for template**
   - Use existing stock as base

---

### 3. Use Template

**Purpose:** Base new stock on existing item.

**Step-by-Step Instructions:**

1. **In manual creation card**

2. **Search for existing stock**
   - Type name

3. **Select from results**
   - Pre-fills form with template data

4. **Modify as needed**

5. **Save as new stock**

---

### 4. Enter Stock Details

**Purpose:** Configure stock item properties.

**Step-by-Step Instructions:**

1. **Enter basic information:**
   - **Name:** Stock item name
   - **Description:** What it is

2. **Select unit type:**
   - Treatments
   - Units
   - ML
   - Grams

3. **Set category (optional)**

4. **Add notes (optional)**

---

### 5. Link to Products

**Purpose:** Connect stock to sellable items.

**Step-by-Step Instructions:**

1. **Find items section**

2. **Search for products/services**

3. **Select items to link**

4. **Stock usage tracked when items sold**

---

### 6. Review AI-Generated Stock

**Purpose:** Verify and correct AI suggestions.

**Step-by-Step Instructions:**

1. **After AI generates items**

2. **Review each suggestion:**
   - Check name accuracy
   - Verify description
   - Confirm category

3. **Edit any incorrect fields**

4. **Remove unwanted suggestions**

5. **Confirm to save**

---

### 7. Save Stock Item

**Purpose:** Create the stock item.

**Step-by-Step Instructions:**

1. **Verify all fields**

2. **Click "Save" button**

3. **Stock created**
   - Added to inventory
   - Ready for tracking

---

### 8. Edit Existing Stock

**Purpose:** Modify existing stock item.

**Step-by-Step Instructions:**

1. **Access via URL with ID**
   - `?id=stock_xxx`

2. **Form pre-fills with existing data**

3. **Make changes**

4. **Save updates**

---

## Creation Methods

| Method | Best For |
|--------|----------|
| AI | Quick entry from photos or descriptions |
| Manual | Precise control over all fields |
| Template | Similar items to existing stock |

---

## Stock Unit Types

| Unit | Use Case |
|------|----------|
| Treatments | Service-based items (e.g., injections) |
| Units | Discrete countable items |
| ML | Liquid products |
| Grams | Weight-based products |
