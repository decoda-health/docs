# Purchase Orders

Create and manage purchase orders for restocking inventory. Track orders from creation through delivery.

<Steps>
  <Step title="Navigate to Purchase Orders">

    View draft, pending, received, or cancelled orders
  </Step>
</Steps>

## Create Purchase Order

<Steps>
  <Step title="Click Create Order">
    Opens order form
  </Step>
  <Step title="Select supplier">
    Choose or add vendor
  </Step>
  <Step title="Add items">
    Select products and quantities
  </Step>
  <Step title="Set details">
    Expected date, notes
  </Step>
  <Step title="Save as draft or submit">
    Order created
  </Step>
</Steps>

## Order Status

| Status | Description |
|--------|-------------|
| **Draft** | Not yet submitted |
| **Pending** | Awaiting delivery |
| **Partial** | Some items received |
| **Received** | All items received |
| **Cancelled** | Order cancelled |

## Receive Order

<Steps>
  <Step title="Open order">
    Click on pending order
  </Step>
  <Step title="Click Receive">
    Opens receiving form
  </Step>
  <Step title="Enter received quantities">
    Mark items as received
  </Step>
  <Step title="Save">
    Stock levels update automatically
  </Step>
</Steps>

<Info>
Partial receiving is supported. Receive items as they arrive.
</Info>

## Troubleshooting

| Issue | Solution |
|-------|----------|
| Can't receive | Check order is in Pending status |
| Stock not updating | Verify receiving was saved |
| Items missing | Check product exists in catalog |
