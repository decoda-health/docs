# Edit Membership

Configure membership details including pricing, billing frequency, and included benefits.

## Basic Details

<Steps>
  <Step title="Enter name">
    Display name for the membership
  </Step>
  <Step title="Add description">
    Explain what's included
  </Step>
  <Step title="Set price">
    Monthly or periodic cost
  </Step>
  <Step title="Choose frequency">
    Monthly, quarterly, semi-annually, or annually
  </Step>
</Steps>

## Configure Benefits

### Included Services

<Steps>
  <Step title="Click Add Service">
    Opens service selector
  </Step>
  <Step title="Select services">
    Choose which services are included
  </Step>
  <Step title="Set discount">
    Percentage off for each service
  </Step>
</Steps>

### Patient Credits

<Steps>
  <Step title="Enable credits">
    Toggle credit allocation
  </Step>
  <Step title="Set amount">
    Credit amount per billing period
  </Step>
  <Step title="Configure rollover">
    Whether unused credits carry forward
  </Step>
</Steps>

### Global Discount

Set a percentage discount that applies to all purchases for members.

## Save Membership

<Steps>
  <Step title="Review settings">
    Verify all configurations
  </Step>
  <Step title="Click Save">
    Changes apply immediately
  </Step>
</Steps>

<Warning>
Changes to existing memberships affect current members on their next billing cycle.
</Warning>

## Troubleshooting

| Issue | Solution |
|-------|----------|
| Can't save | Check required fields are filled |
| Service not available | Verify service is active |
| Price rejected | Check valid currency format |
