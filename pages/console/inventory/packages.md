# Packages Page - User Actions Guide

## Overview

The Packages page manages product/service bundles sold at bundled prices. Staff can create, edit, and delete packages, view usage statistics, and manage pricing and categories.

---

## Available Actions

### 1. View All Packages

**Purpose:** Browse all available packages.

**Step-by-Step Instructions:**

1. **Navigate to Packages**
   - Inventory menu → Packages

2. **View package list**
   - Name and description
   - Category
   - Price
   - Active/all-time patient count

---

### 2. Add New Package

**Purpose:** Create a new product bundle.

**Step-by-Step Instructions:**

1. **Click "Add Package" button**
   - Top right corner

2. **Enter package details:**
   - Name
   - Description
   - Price
   - Category
   - Internal notes

3. **Add items to package**
   - Select products/services
   - Set quantities

4. **Save package**

---

### 3. Edit Package

**Purpose:** Modify existing package.

**Step-by-Step Instructions:**

1. **Find package in list**

2. **Click menu (three dots)**

3. **Select "Edit and view details"**

4. **Modify details**
   - Update name, price, items

5. **Save changes**

---

### 4. Inline Edit Category

**Purpose:** Quickly change package category.

**Step-by-Step Instructions:**

1. **Click on category cell**
   - Opens popover

2. **Select category from dropdown**
   - Or "No category"

3. **Category updates immediately**

---

### 5. Inline Edit Price

**Purpose:** Quickly change package price.

**Step-by-Step Instructions:**

1. **Click on price cell**
   - Opens popover

2. **Enter new price**
   - Or press Enter

3. **Price updates immediately**

---

### 6. View Package Usage

**Purpose:** See patients who purchased package.

**Step-by-Step Instructions:**

1. **Click patient count link**
   - Shows "X active"

2. **Redirected to usage page**

---

### 7. Delete Package

**Purpose:** Remove a package.

**Step-by-Step Instructions:**

1. **Click menu (three dots)**

2. **Select "Delete"**

3. **Confirm deletion**
   - Active patients keep access
   - No new purchases

---

### 8. Group by Category

**Purpose:** Organize packages by category.

**Step-by-Step Instructions:**

1. **Click "Group by" dropdown**
   - Appears if packages have categories

2. **Select "Category"**

3. **Packages grouped by category**
   - Collapsible sections

---

### 9. Filter Packages

**Purpose:** Find specific packages.

**Step-by-Step Instructions:**

1. **Use filter controls**
   - By category
   - By search term

2. **Table updates**

---

### 10. Sort Packages

**Purpose:** Organize package list.

**Step-by-Step Instructions:**

1. **Click column header**

2. **Toggle sort direction**

---

## Package Table Columns

| Column | Description |
|--------|-------------|
| Package | Name, description, internal notes |
| Category | Package category |
| Price | Bundle price |
| Patients | Active and all-time count |
| Actions | Menu options |

---

## Package Contents

Each package contains:
- One or more items/services
- Quantity per item
- Total bundle price (not sum of items)

| View packages | ✓ | ✓ | ✓ |

| Edit packages | ✓ | ✓ | ✗ |
| Delete packages | ✓ | ✓ | ✗ |

## Troubleshooting

| Issue | Solution |
|-------|----------|
| Can't delete | Check if has active users |
| Category not showing | Add categories first |
| Price not updating | Check valid number format |
| No packages | Create first package |
