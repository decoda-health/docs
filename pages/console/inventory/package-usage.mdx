# Package Usage

Track patient package purchases and usage. View remaining credits, usage history, and manage allocations.

## View Usage

<Steps>
  <Step title="Access from Packages">
    Click patient count on a package row
  </Step>
  <Step title="Browse patients">
    See all patients who purchased this package
  </Step>
  <Step title="View details">
    Click a patient to see their usage
  </Step>
</Steps>

## Usage Details

Each patient's package shows:
- Purchase date
- Total credits/sessions
- Used credits
- Remaining balance
- Expiration date (if applicable)

## Apply Package Credit

When a patient uses a service from their package:

<Steps>
  <Step title="Go to checkout">
    Start a charge for the patient
  </Step>
  <Step title="Add service">
    Select the included service
  </Step>
  <Step title="Apply package">
    System suggests package credit
  </Step>
  <Step title="Complete">
    Credit deducted from package
  </Step>
</Steps>

<Tip>
Package credits are automatically suggested during checkout when applicable.
</Tip>

## Troubleshooting

| Issue | Solution |
|-------|----------|
| Credits not applying | Check service matches package items |
| Package expired | Credits no longer available |
| Usage incorrect | Review transaction history |
