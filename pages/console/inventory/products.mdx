# Products

The Products page manages your practice's product catalog including retail items, consumables, and equipment. Add, edit, categorize, and track inventory levels.

## View Products

<Steps>
  <Step title="Navigate to Products">
    Go to **Inventory** â†’ **Products**
  </Step>
  <Step title="Browse catalog">
    View name, SKU, price, stock level, and category for each product
  </Step>
  <Step title="Search and filter">
    Use search box or category filter to find products
  </Step>
</Steps>

## Add Product

<Steps>
  <Step title="Click Add Product">
    Button in the header
  </Step>
  <Step title="Fill product details">
    - **Name**: Display name
    - **SKU**: Unique stock keeping unit
    - **Description**: Product details
    - **Category**: Assign to category
    - **Price**: Selling price
    - **Cost**: Your purchase cost
    - **Tax**: Tax category
    - **Image**: Upload product photo
  </Step>
  <Step title="Configure inventory">
    Toggle inventory tracking, set initial stock and reorder threshold
  </Step>
  <Step title="Save">
    Product added to catalog
  </Step>
</Steps>

## Edit Product

Click on a product or the edit icon to open the edit form. Modify any details and save.

## Manage Stock

<Steps>
  <Step title="Find product">
    Locate in the list
  </Step>
  <Step title="View current stock">
    Stock level displayed on the row
  </Step>
  <Step title="Adjust stock">
    Click adjust button, enter new quantity or +/- adjustment
  </Step>
  <Step title="Record reason">
    Note the reason: sale, restock, damage, etc.
  </Step>
</Steps>

## Low Stock Alerts

<Steps>
  <Step title="Edit product">
    Open product settings
  </Step>
  <Step title="Set reorder threshold">
    Enter minimum quantity
  </Step>
  <Step title="Save">
    Alert triggers when stock falls below threshold
  </Step>
</Steps>

<Warning>
Products with active transactions cannot be deleted. Deactivate instead to hide from sales.
</Warning>

## Product Fields

| Field | Description |
|-------|-------------|
| Name | Display name |
| SKU | Stock keeping unit (unique) |
| Description | Detailed description |
| Category | Product category |
| Price | Selling price |
| Cost | Your cost |
| Tax Category | Tax treatment |
| Stock | Current quantity |
| Threshold | Low stock alert level |

## Actions

| Action | Description |
|--------|-------------|
| **Edit** | Modify product details |
| **Deactivate** | Hide from sales (preserves history) |
| **Delete** | Permanently remove |
| **Export** | Download catalog as CSV/Excel |

## Troubleshooting

| Issue | Solution |
|-------|----------|
| Product not showing | Check if active, check category filter |
| Image not uploading | Check file size (max 5MB) and format (JPG, PNG, WebP) |
| Stock incorrect | Review adjustment history |
| Can't delete | Product has associated transactions |
