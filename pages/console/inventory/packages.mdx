# Packages

Manage product/service bundles sold at bundled prices. Create, edit, and delete packages, view usage statistics, and manage pricing.

## View Packages

<Steps>
  <Step title="Navigate to Packages">
    Go to **Inventory** → **Packages**
  </Step>
  <Step title="Browse package list">
    See name, category, price, and patient count
  </Step>
  <Step title="Group by category">
    Use the "Group by" dropdown for organized view
  </Step>
</Steps>

## Create Package

<Steps>
  <Step title="Click Add Package">
    Button in top right corner
  </Step>
  <Step title="Enter details">
    Name, description, price, category, internal notes
  </Step>
  <Step title="Add items">
    Select products/services and set quantities
  </Step>
  <Step title="Save">
    Package available for purchase
  </Step>
</Steps>

## Quick Edit

### Inline Category Edit

Click on the category cell → select from dropdown → changes save immediately.

### Inline Price Edit

Click on the price cell → enter new price → press Enter → changes save immediately.

## View Usage

Click the patient count link (shows "X active") to see patients who purchased the package.

## Delete Package

<Steps>
  <Step title="Open menu">
    Click three dots on the package row
  </Step>
  <Step title="Select Delete">
    Confirm the action
  </Step>
</Steps>

<Warning>
Active patients keep access to deleted packages. No new purchases allowed.
</Warning>

## Package Contents

Each package contains:
- One or more items/services
- Quantity per item
- Total bundle price (not sum of individual items)

## Table Columns

| Column | Description |
|--------|-------------|
| Package | Name, description, internal notes |
| Category | Package category |
| Price | Bundle price |
| Patients | Active and all-time count |
| Actions | Menu options |

## Troubleshooting

| Issue | Solution |
|-------|----------|
| Can't delete | Check if has active users |
| Category not showing | Add categories first |
| Price not updating | Check valid number format |
