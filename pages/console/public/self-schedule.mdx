# Self-Schedule

The Self-Schedule page is the patient-facing appointment booking interface. Patients can browse available times, select services, and book appointments.

<Info>
This is a public page accessible without login. Patients receive links via SMS or email.
</Info>

## Booking Flow

<Steps>
  <Step title="Select service">
    Patient chooses from available services
  </Step>
  <Step title="Pick provider (optional)">
    Select preferred provider or any available
  </Step>
  <Step title="Choose date/time">
    View available slots and select one
  </Step>
  <Step title="Enter details">
    Name, phone, email, and any required info
  </Step>
  <Step title="Confirm">
    Appointment booked, confirmation sent
  </Step>
</Steps>

## Features

<CardGroup cols={2}>
  <Card title="Real-time Availability" icon="clock">
    Shows only genuinely available slots
  </Card>
  <Card title="Mobile Friendly" icon="mobile">
    Fully responsive design
  </Card>
</CardGroup>

## Patient Information

Required fields may include:
- First and last name
- Phone number
- Email
- Date of birth
- Insurance information

## Confirmation

After booking, patient receives:
- On-screen confirmation
- SMS confirmation (if opted in)
- Email with appointment details
- Calendar invite option
