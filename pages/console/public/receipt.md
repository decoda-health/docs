# Receipt Page - User Actions Guide

## Overview

The Receipt page displays a payment receipt for a completed transaction. It's accessible to patients without authentication and provides a printable record of their payment.

---

## Available Actions

### 1. View Receipt Details

**Purpose:** Review the complete payment receipt.

**Step-by-Step Instructions:**

1. **Access the receipt**
   - Click receipt link from confirmation email/SMS
   - Or access via payment confirmation page

2. **View receipt sections:**

   **Header:**
   - Practice logo
   - Practice name
   - Receipt/confirmation number

   **Payment Information:**
   - Payment date
   - Payment amount
   - Payment method used (last 4 digits)

   **Patient Information:**
   - Patient name
   - Account number

   **Items/Services:**
   - Description of what was paid for
   - Individual item amounts
   - Subtotal

   **Payment Summary:**
   - Total paid
   - Any tips included
   - Fees (if applicable)

---

### 2. Print Receipt

**Purpose:** Create a physical copy of the receipt.

**Step-by-Step Instructions:**

1. **Open print dialog**
   - Press `Ctrl/Cmd + P`
   - Or use browser menu â†’ Print

2. **Configure print settings:**
   - Select printer
   - Paper size: Letter or A4
   - Enable "Background graphics" for branding

3. **Print the receipt**
   - Click "Print"
   - Receipt prints on selected printer

---

### 3. Save Receipt as PDF

**Purpose:** Download a digital copy of the receipt.

**Step-by-Step Instructions:**

1. **Open print dialog**
   - Press `Ctrl/Cmd + P`

2. **Select PDF option:**
   - Destination: "Save as PDF"
   - Or "Microsoft Print to PDF"

3. **Save the file**
   - Choose location
   - Enter filename
   - Click "Save"

---

### 4. Share Receipt

**Purpose:** Send the receipt to someone else.

**Step-by-Step Instructions:**

1. **Copy the URL**
   - Copy the page URL from browser
   - URL contains payment ID

2. **Share the link**
   - Email, text, or message the URL
   - Recipient can view the receipt

---

## Receipt Components

| Component | Description |
|-----------|-------------|
| Logo | Practice branding |
| Receipt Number | Unique identifier |
| Date | When payment was made |
| Patient Name | Who paid |
| Amount | Total payment |
| Method | How it was paid |
| Items | What was paid for |

---

## Authentication Note

This receipt page is accessible without authentication:
- Anyone with the link can view it
- Receipt ID acts as the authentication
- Suitable for sharing with insurance, employers, etc.

| Issue | Solution |
|-------|----------|

| Can't save PDF | Try different browser |
