# Usage & Billing

The Usage page provides transparency into your Decoda platform usage and billing. View usage metrics, costs, historical statements, and manage payment methods.

## View Current Usage

<Steps>
  <Step title="Navigate to Usage">
    Click **Usage** in the navigation menu
  </Step>
  <Step title="Review summary">
    Total estimated charges displayed with category breakdown
  </Step>
  <Step title="View usage chart">
    Visual representation showing usage trends over the month
  </Step>
</Steps>

## Select Billing Period

<Steps>
  <Step title="Open month selector">
    Click the dropdown in the page header (shows current month by default)
  </Step>
  <Step title="Choose month">
    Select from current or previous months
  </Step>
  <Step title="View data">
    Usage chart and table update with selected period data
  </Step>
</Steps>

## Usage Breakdown

The itemized table shows usage by feature:

| Category | What's Measured |
|----------|-----------------|
| Platform Access | Base platform fee |
| Appointments | Number of appointments scheduled |
| SMS Messages | Outbound text messages sent |
| Phone Calls | Call minutes used |
| AI Scribe | Transcription minutes |
| AI Phone | AI call handling minutes |
| Storage | Document and image storage |
| API Calls | Integration usage |

<Tip>
Hover over chart data points to see exact daily values and costs.
</Tip>

## View Statements

<Steps>
  <Step title="Locate statements section">
    Find the **Statements** table on the page
  </Step>
  <Step title="Review statement info">
    See invoice date, period covered, total amount, and status
  </Step>
  <Step title="Download statement">
    Click the download icon for PDF version
  </Step>
</Steps>

## Manage Payment Methods

<Steps>
  <Step title="Click Manage Payments">
    Button in the page header opens payment modal
  </Step>
  <Step title="View current cards">
    See saved payment methods with last 4 digits and expiration
  </Step>
  <Step title="Add new card">
    Enter card number, expiration, CVV, and billing ZIP
  </Step>
  <Step title="Set default">
    Click **Set as Default** on the card to use for automatic billing
  </Step>
</Steps>

<Warning>
You cannot remove a payment method if it's the only one on file.
</Warning>

## Free Usage Allocation

<Note>
Your plan includes free monthly allocations for SMS, AI minutes, and storage. Usage within these limits incurs no additional charge. Overages are billed at the displayed rates.
</Note>

Look for the "included monthly usage" note below the usage table. Hover for details on:
- Free SMS messages
- Included AI minutes
- Included storage allocation

## Billing Cycle

- **Monthly billing**: Invoiced at end of each month
- **Payment timing**: Charged to default payment method
- **Statement availability**: Within first week of new month

<Info>
Usage estimates are for planning purposes. Final invoices may differ due to rounding and taxes.
</Info>

## Troubleshooting

| Issue | Solution |
|-------|----------|
| Usage not loading | Refresh page, check internet connection |
| Statement missing | Allow a few days after period ends |
| Payment failed | Update payment method, verify card details |
| Usage seems high | Review itemized breakdown by category |
| Can't add card | Verify card number and details are correct |
